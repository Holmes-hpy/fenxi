## 一、分析框架

### 1.1 财务分析框架

**杜邦分析体系**：
```
ROE = 净利率 × 资产周转率 × 权益乘数

     = (净利润/收入) × (收入/资产) × (资产/净资产)

     = 销售净利率 × 资产周转率 × 权益乘数
```

**优秀公司的财务特征**：
- ROE > 15%，且稳定
- 净利润率 > 10%
- 资产周转率 > 0.8
- 资产负债率适中（< 60%）

### 1.2 估值方法

**相对估值法**：
| 方法 | 适用场景 | 优点 | 缺点 |
|-----|---------|------|------|
| PE估值 | 盈利稳定的公司 | 简单直观 | 受行业影响大 |
| PB估值 | 金融、重资产公司 | 稳定 | 不适合轻资产 |
| PS估值 | 成长初期公司 | 避免亏损干扰 | 不考虑盈利 |
| EV/EBITDA | 资本密集型 | 跨行业可比 | 计算复杂 |

**绝对估值法**：
- **现金流折现法（DCF）**
- 折现率：WACC（加权平均资本成本）
- 永续增长率：假设公司永续增长率

### 1.3 护城河分析

**五种护城河**：

1. **无形资产**
   - 品牌（茅台）
   - 专利（药企）
   - 许可证（金融、电信）

2. **转换成本**
   - 软件用户习惯
   - 工业配套
   - 金融服务

3. **网络效应**
   - 社交网络（微信）
   - 电商平台（淘宝）
   - 操作系统（Windows）

4. **成本优势**
   - 地理优势
   - 流程优势
   - 地理位置
   - 规模经济

5. **有效规模**
   - 市场容量有限
   - 新进入者无利可图

## 三、公司分析清单

### 3.1 定性分析

| 项目 | 分析要点 |
|-----|---------|
| **商业模式** | 如何赚钱？靠什么盈利？ |
| **行业空间** | 市场有多大？增长多快？ |
| **竞争格局** | 行业地位如何？竞争对手是谁？ |
| **护城河** | 有哪些竞争优势？能持续多久？ |
| **管理层** | 诚信度？能力？激励机制？ |
| **公司治理** | 股权结构？激励机制？关联交易？ |

### 3.2 定量分析

| 项目 | 分析要点 | 合格线 |
|-----|---------|--------|
| **盈利能力** | ROE、毛利率、净利率 | ROE>15% |
| **成长能力** | 营收增速、利润增速 | 营收增长>10% |
| **运营能力** | 存货周转、应收账款周转 | 周转加快 |
| **偿债能力** | 资产负债率、流动比率 | 负债率<60% |
| **现金流** | 经营现金流/净利润 | >1 |
| **分红能力** | 分红率、股息率 | 稳定分红 |

### 3.3 估值分析

| 项目 | 分析要点 |
|-----|---------|
| **绝对估值** | DCF估值、内在价值 |
| **相对估值** | PE/PB/PS与行业平均、历史水平比较 |
| **安全边际** | 当前价格 vs 内在价值 |
| **估值驱动** | 什么因素能让估值提升？ |

## 四、投资决策清单

### 4.1 买入前必查

- [ ] 主营业务是什么？靠什么赚钱？
- [ ] 行业空间有多大？增长逻辑是什么？
- [ ] 竞争优势是什么？能否持续？
- [ ] 财务数据是否健康？
- [ ] 估值是否合理？安全边际够不够？
- [ ] 管理层是否诚信有能力？
- [ ] 有哪些潜在风险？

### 4.2 持有期跟踪

- [ ] 季度财报是否符合预期？
- [ ] 竞争优势是否被削弱？
- [ ] 行业格局是否变化？
- [ ] 估值是否出现泡沫？
- [ ] 是否有新的风险因素？

### 4.3 卖出信号

| 信号类型 | 说明 |
|---------|------|
| **基本面恶化** | 竞争优势丧失、业绩持续下滑 |
| **估值泡沫** | 估值显著高于合理水平 |
| **发现更好的机会** | 有更具性价比的投资标的 |
| **系统性风险** | 宏观经济或市场出现重大风险 |
| **判断错误** | 买入逻辑被证伪 |
