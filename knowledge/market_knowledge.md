- 长度：40个月左右

## 二、资金流向规律

### 2.1 北向资金分析

**北向资金构成**：
- 合格境外机构投资者（QFII）
- 人民币合格境外机构投资者（RQFII）
- 沪港通、深港通

**北向资金偏好**：
- 大盘蓝筹股
- 行业龙头
- 估值合理的成长股

### 2.2 主力资金分类

| 类型 | 特点 | 操作风格 |
|-----|------|---------|
| **公募基金** | 规模大、持仓周期长 | 价值投资、组合投资 |
| **私募基金** | 灵活、追求绝对收益 | 趋势投资、主题投资 |
| **社保基金** | 追求稳健、长期持有 | 价值投资、底部布局 |
| **保险资金** | 规模大、期限长 | 价值投资、大类资产配置 |
| **游资** | 灵活、快速 | 题材炒作、短线操作 |

### 2.3 资金流向判断

**资金净流入判断方法**：
```
资金净流入 = 主买金额 - 主卖金额

- 净流入 > 0：资金流入
- 净流入 < 0：资金流出
```

## 三、周期分析方法

### 3.1 经济周期定位

**复苏期**：
- 特征：GDP增速回升、PMI站上50、CPI低位
- 受益板块：可选消费、金融、原材料
- 投资策略：增加股票配置，减少债券

**繁荣期**：
- 特征：GDP高速增长、通胀抬头、货币政策收紧
- 受益板块：周期股、涨价题材
- 投资策略：持有周期股，准备减仓

**衰退期**：
- 特征：GDP增速下滑、PMI跌破50通缩风险
- 受益板块：防御性板块（医药、消费）
- 投资策略：增加现金配置，持有防御股

**萧条期**：
- 特征：GDP负增长、失业率上升、货币政策宽松
- 受益板块：政策刺激受益板块
- 投资策略：逢低布局，等待复苏

### 3.2 行业轮动规律

**美林投资时钟**：
```
经济下行、通胀下行 → 衰退期 → 债券
经济上行、通胀下行 → 复苏期 → 股票
经济上行、通胀上行 → 过热期 → 商品
经济下行、通胀上行 → 滞胀期 → 现金
```

**A股行业轮动**：
- 政策出台 → 预期改善 → 金融股启动
- 经济复苏 → 需求增加 → 周期股启动
- 消费升级 → 品牌崛起 → 消费股启动
- 技术进步 → 新兴产业 → 成长股启动

## 四、市场异常现象

### 4.1 常见市场规律

| 现象 | 规律 | 应用 |
|-----|------|------|
| **月末效应** | 月末市场往往波动加大 | 注意风险控制 |
| **节假日效应** | 节假日前后市场成交清淡 | 观望为主 |
| **财报季效应** | 财报发布前后股价异动 | 关注业绩公告 |
| **政策市特征** | 重大政策出台引发市场波动 | 关注政策动向 |

### 4.2 技术分析规律

| 规律 | 说明 |
|-----|------|
| **支撑位与压力位** | 价格在某个位置反复遇到买盘或卖盘 |
| **均线支撑压力** | MA5、MA10、MA20、MA60形成支撑/压力 |
| **突破与回踩** | 突破后往往回踩确认 |
| **背离现象** | 价格与指标背离往往预示转折 |

### 4.3 行为金融学规律

| 现象 | 描述 | 应对 |
|-----|------|------|
| **羊群效应** | 投资者盲目跟随他人 | 独立思考 |
| **损失厌恶** | 亏损的痛苦大于盈利的快乐 | 严格执行止损 |
| **过度自信** | 高估自己的判断能力 | 保持谦逊 |

| **低位背离** | 价格创新低，指标未跟随 |
| **利空不跌** | 坏消息出台股价不跌 |
| **恐慌抛售** | 投资者恐慌性卖出，往往是底部 |
