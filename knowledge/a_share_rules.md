# 📋 A股交易制度与规则知识库

---

## 一、交易规则

### 1.1 交易时间

| 时段 | 时间 | 说明 |
|------|------|------|
| 集合竞价（早盘） | 9:15-9:25 | 其中9:15-9:20可撤单，9:20-9:25不可撤单 |
| 连续竞价 | 9:30-11:30 | 正常交易时段 |
| 午间休市 | 11:30-13:00 | 不可交易 |
| 连续竞价 | 13:00-14:57 | 正常交易时段 |
| 集合竞价（尾盘） | 14:57-15:00 | 不可撤单 |
| 盘后定价交易 | 15:05-15:30 | 仅科创板、创业板 |

### 1.2 T+1交易制度
- **定义**：当天买入的股票，下一个交易日才能卖出
- **例外**：ETF、LOF等基金产品可能支持T+0
- **影响**：限制短线交易，增加市场稳定性

### 1.3 涨跌幅限制

| 板块 | 涨跌幅限制 | 备注 |
|------|-----------|------|
| 主板（沪市、深市） | ±10% | ST/*ST股票±5% |
| 科创板 | ±20% | 新股上市前5个交易日不设涨跌幅 |
| 创业板 | ±20% | 新股上市前5个交易日不设涨跌幅 |
| 北交所 | ±30% | 新股上市首日不设涨跌幅 |
| 风险警示板 | ±5% | ST/*ST股票 |

### 1.4 停牌规则

#### 临时停牌
- **异常波动停牌**：连续3个交易日内涨跌幅累计达到±20%、±15%（ST股票）
- **核查停牌**：公司需要核查市场传闻或异常交易
- **信息披露停牌**：重要事项未披露前

#### 长期停牌
- **重大资产重组**：最长不超过10个交易日，可申请延期
- **退市停牌**：被实施退市风险警示或终止上市

### 1.5 异常波动监控
- **涨跌幅偏离值**：单只股票涨跌幅与对应指数涨跌幅的差值
- **换手率异常**：连续多日换手率过高
- **龙虎榜**：上榜条件包括涨跌幅、换手率、振幅等

## 二、发行与退市制度

### 2.1 IPO审核流程

#### 注册制（科创板、创业板、北交所）
1. **申报**：公司向交易所提交申请文件
2. **问询**：交易所审核，提出问询
3. **注册**：证监会注册
4. **发行**：询价、定价、申购、上市

#### 核准制（主板）
1. **辅导**：券商辅导，验收通过
2. **申报**：向证监会提交申请
3. **审核**：发审委审核
4. **发行**：核准后发行

### 2.2 注册制改革要点
- **市场化定价**：取消23倍市盈率限制
- **信息披露为核心**：强调真实、准确、完整披露
- **压实中介责任**：券商、会计师、律师责任加重

### 2.3 退市条件

#### 财务类退市（最常见）
| 指标 | 退市条件 | 风险警示条件 |
|------|---------|-------------|
| 净利润 | 连续2年为负且营收低于1亿 | 连续1年为负且营收低于1亿 |
| 净资产 | 连续2年为负 | 连续1年为负 |
| 审计意见 | 连续2年非标意见 | 连续1年非标意见 |
| 营业收入 | 连续2年低于1亿 | 连续1年低于1亿 |

#### 交易类退市
- 连续20个交易日收盘价低于1元（面值退市）
- 连续20个交易日市值低于3亿
- 连续20个交易日股东人数少于2000人

#### 规范类退市
- 未按时披露年报/半年报
- 信息披露或规范运作存在重大问题

### 2.4 退市风险警示（ST/*ST）
- **ST**：其他风险警示（财务状况异常等）
- ***ST**：退市风险警示（面临退市）
- **影响**：涨跌幅限制变为±5%

## 三、监管体系

### 3.1 监管机构分工

| 机构 | 职责 |
|------|------|
| **证监会（CSRC）** | 统一监管全国证券市场，制定规则 |
| **上交所（SSE）** | 沪市交易监管，上市审核（科创板） |
| **深交所（SZSE）** | 深市交易监管，上市审核（创业板） |
| **北交所（BSE）** | 北交所交易监管，上市审核 |
| **中证登（CSDC）** | 证券登记、存管、清算 |

### 3.2 常见监管措施

#### 行政监管措施
- **问询函**：要求公司解释特定问题
- **监管谈话**：口头警示
- **出具警示函**：书面警示
- **责令改正**：要求限期整改

#### 行政处罚
- **罚款**：对违法违规行为罚款
- **市场禁入**：禁止进入证券市场
- **撤销资格**：撤销相关业务资格

#### 交易限制
- **限制交易**：限制特定账户交易
- **暂停账户交易**：暂停账户一段时间

### 3.3 内幕交易与操纵市场
- **内幕交易**：利用未公开信息交易
- **操纵市场**：通过虚假交易、对倒等手段影响价格
- **处罚**：罚款、市场禁入、刑事责任

## 四、特殊板块规则

### 4.1 科创板
- **准入条件**：开通前20个交易日日均资产≥50万，交易经验≥2年
- **交易规则**：±20%涨跌幅，盘后定价交易
- **特点**：面向科技创新企业，允许未盈利企业上市

### 4.2 创业板
- **准入条件**：开通前20个交易日日均资产≥10万，交易经验≥2年
- **交易规则**：±20%涨跌幅，盘后定价交易
- **特点**：面向成长型创新创业企业

### 4.3 北交所
- **准入条件**：开通前20个交易日日均资产≥50万，交易经验≥2年
- **交易规则**：±30%涨跌幅，首日不设涨跌幅
- **特点**：服务创新型中小企业

### 4.4 风险警示板
- **股票范围**：ST/*ST股票
- **交易限制**：单日累计买入数量限制
- **涨跌幅**：±5%

## 五、分红与再融资

### 5.1 分红政策
- **现金分红**：以现金形式分配利润
- **送股**：以股票形式分配利润（股本增加）
- **转增股本**：将资本公积转为股本

#### 分红条件（主板）
- 最近3年累计现金分红不少于年均可分配利润的30%

### 5.2 再融资方式

| 方式 | 说明 | 特点 |
|------|------|------|
| **定增** | 向特定投资者增发股票 | 锁定期6个月-3年 |
| **配股** | 向原股东配售股票 | 优先配售权 |
| **可转债** | 可转换为股票的债券 | 兼具债性和股性 |
| **优先股** | 优先分配股息的股票 | 无投票权 |

### 5.3 股权激励
- **股票期权**：未来以约定价格购买股票的权利
- **限制性股票**：有锁定期和解锁条件的股票
- **影响**：可能稀释股东权益，但激励管理层

## 六、与海外市场的主要差异

| 项目 | A股 | 海外市场（美股） |
|------|-----|-----------------|
| 交易制度 | T+1 | T+0 |
| 涨跌幅 | 有（10%-30%） | 无 |
| 做空机制 | 受限 | 成熟 |
| IPO定价 | 曾受限制，逐步市场化 | 完全市场化 |
| 退市标准 | 多维度，较严格 | 严格 |
| 投资者结构 | 散户为主 | 机构为主 |

## 七、投资者常见踩坑点

### 7.1 规则陷阱
1. **T+1误操作**：当天买入想卖出无法成交
2. **ST股票风险**：涨跌幅小且可能退市
3. **停牌期间无法交易**：资金被锁定
4. **盘后定价交易规则**：申报时间和价格限制

### 7.2 监管风险
1. **问询函解读**：负面问询可能导致股价下跌
2. **监管处罚**：直接影响公司估值
3. **立案调查**：重大利空信号

### 7.3 退市风险
1. **面值退市**：股价低于1元需警惕
2. **财务退市**：连续亏损需关注
3. **审计意见**：非标意见是预警信号
