#### 产业链价值分布

| 产业链环节 | 特点 | 利润空间 | 代表行业 |
|-----------|------|---------|---------|
| **上游** | 资源型、周期性 | 波动大 | 煤炭、有色、化工原料 |
| **中游** | 加工制造 | 取决于技术壁垒 | 零部件、设备制造 |
| **下游** | 品牌渠道 | 品牌溢价高 | 消费品、终端产品 |

### 1.2 产业链传导机制

#### 价格传导
- **上游涨价 → 中游成本上升 → 下游提价压力**
- **传导时滞**：不同行业传导速度不同
- **议价能力**：决定利润分配

#### 需求传导
- **下游需求增加 → 中游产能扩张 → 上游订单增加**
- **库存周期**：影响传导节奏

#### 技术传导
- **上游技术突破 → 中游工艺改进 → 下游产品升级**
- **技术扩散速度**：决定行业变革速度

### 1.3 关键环节识别

#### 判断标准
1. **技术壁垒**：是否有核心技术专利
2. **资本壁垒**：是否需要大量资本投入
3. **规模效应**：是否存在规模经济
4. **转换成本**：客户是否容易转换供应商
5. **网络效应**：是否具有网络效应

#### 关键环节特征
- **高毛利率**：持续高于行业平均
- **高ROE**：持续高于行业平均
- **定价权**：能够自主定价
- **护城河**：难以被替代

## 二、行业生命周期分析

### 2.1 生命周期四阶段

| 阶段 | 特征 | 投资策略 | 风险 |
|------|------|---------|------|
| **导入期** | 技术不成熟，需求小 | 风险投资，高风险高回报 | 技术失败、市场接受度 |
| **成长期** | 技术成熟，需求爆发 | 积极投资，分享增长 | 竞争加剧、估值过高 |
| **成熟期** | 需求稳定，竞争激烈 | 价值投资，关注分红 | 增长放缓、边际改善 |
| **衰退期** | 需求下降，产能过剩 | 谨慎投资，寻找转型机会 | 业绩下滑、退市风险 |

### 2.2 生命周期判断指标

```python
def judge_lifecycle(industry_data):
    """
    判断行业生命周期阶段
    
    主要指标：
    1. 行业增速
    2. 渗透率
    3. 竞争格局
    4. 技术创新速度
    5. 政策支持力度
    """
    indicators = {
        'growth_rate': industry_data.get('growth_rate', 0),
        'penetration_rate': industry_data.get('penetration_rate', 0),
        'concentration_ratio': industry_data.get('concentration_ratio', 0),
        'tech_innovation': industry_data.get('tech_innovation', 0),
        'policy_support': industry_data.get('policy_support', 0)
    }
    
    if indicators['growth_rate'] > 30 and indicators['penetration_rate'] < 20:
        return "导入期"
    elif indicators['growth_rate'] > 15 and indicators['penetration_rate'] < 50:
        return "成长期"
    elif 5 <= indicators['growth_rate'] <= 15 and indicators['penetration_rate'] >= 50:
        return "成熟期"
    else:
        return "衰退期"
```

### 2.3 不同阶段的投资策略

#### 导入期
- **投资对象**：技术领先的初创公司
- **策略**：分散投资，高风险高回报
- **退出机制**：IPO或被收购

#### 成长期
- **投资对象**：行业龙头，有技术壁垒的公司
- **策略**：集中投资，分享行业增长
- **关注指标**：营收增速、市场份额

#### 成熟期
- **投资对象**：现金流稳定、高分红的公司
- **策略**：价值投资，关注估值
- **关注指标**：PE、PB、分红率

#### 衰退期
- **投资对象**：有转型潜力的公司
- **策略**：谨慎投资，快进快出
- **关注指标**：转型进展、资产质量

## 三、行业竞争格局分析

### 3.1 波特五力模型

#### 五力分析框架

| 力量 | 分析要点 | 影响 |
|------|---------|------|
| **潜在进入者威胁** | 进入壁垒高低 | 决定行业竞争激烈程度 |
| **替代品威胁** | 替代可能性 | 决定行业天花板 |
| **供应商议价能力** | 供应商集中度 | 影响成本 |
| **购买者议价能力** | 购买者集中度 | 影响定价 |
| **现有竞争者竞争** | 竞争方式、差异化 | 影响利润率 |

#### 五力模型应用

```python
def porter_five_forces(industry):
    """
    波特五力分析
    
    返回各力量强度评分（1-5分，5分为最强）
    """
    forces = {
        'potential_entrant': 0,   # 潜在进入者威胁
        'substitute': 0,          # 替代品威胁
        'supplier_power': 0,      # 供应商议价能力
        'buyer_power': 0,         # 购买者议价能力
        'competitor_rivalry': 0   # 现有竞争者竞争
    }
    
    # 根据行业特征评分
    # 示例：半导体行业
    if industry == '半导体':
        forces['potential_entrant'] = 2  # 壁垒高
        forces['substitute'] = 1         # 替代品少
        forces['supplier_power'] = 4     # 设备供应商集中
        forces['buyer_power'] = 3        # 下游集中
        forces['competitor_rivalry'] = 4 # 竞争激烈
    
    return forces
```

### 3.2 行业集中度分析

#### 集中度指标
- **CR4**：前4家企业市场份额
- **CR10**：前10家企业市场份额
- **HHI指数**：赫芬达尔-赫希曼指数

#### 集中度类型

| 类型 | CR4范围 | 特点 | 投资策略 |
|------|--------|------|---------|
| **分散竞争** | <40% | 竞争激烈，无明显龙头 | 寻找潜在龙头 |
| **寡头竞争** | 40%-70% | 少数龙头主导 | 投资龙头 |
| **垄断竞争** | >70% | 一家或几家垄断 | 关注垄断利润 |

### 3.3 龙头公司分析

#### 龙头公司特征
- **市场份额领先**：CR1或CR2
- **盈利能力强**：ROE高于行业平均
- **现金流稳定**：经营现金流充足
- **研发投入高**：持续创新
- **品牌价值高**：客户认可度高

#### 护城河分析
| 护城河类型 | 特点 | 案例 |
|-----------|------|------|
| **品牌** | 消费者心智占领 | 茅台、可口可乐 |
| **技术** | 专利保护、技术领先 | 华为、宁德时代 |
| **规模** | 规模经济、成本优势 | 比亚迪、隆基绿能 |
| **网络效应** | 用户越多价值越大 | 腾讯、美团 |
| **转换成本** | 客户难以转换 | 微软Office |

## 四、政策影响分析

### 4.1 政策类型与影响

#### 政策分类

| 政策类型 | 影响方式 | 典型案例 |
|---------|---------|---------|
| **产业支持政策** | 补贴、税收优惠、研发支持 | 新能源汽车补贴 |
| **行业监管政策** | 准入限制、合规要求 | 医药集采 |
| **宏观经济政策** | 货币政策、财政政策 | 降息降准 |
| **国际贸易政策** | 关税、进出口限制 | 中美贸易战 |

#### 政策影响传导路径

```
政策出台 → 行业预期改变 → 资金流向变化 → 股价波动 → 基本面变化
```

### 4.2 政策解读方法

#### 关键要素
1. **政策目标**：政策要解决什么问题
2. **覆盖范围**：哪些行业/公司受益
3. **执行力度**：政策能否有效执行
4. **时间周期**：政策影响的时间跨度
5. **边际变化**：政策比预期更好还是更差

#### 政策影响评估

```python
def evaluate_policy_impact(policy, industry):
    """
    评估政策对行业的影响
    
    返回：影响方向（正面/中性/负面）、影响程度（高/中/低）
    """
    impact_direction = '中性'
    impact_level = '低'
    
    # 根据政策类型评估
    if policy['type'] == '产业支持':
        if industry in policy['beneficiaries']:
            impact_direction = '正面'
            impact_level = '高'
    elif policy['type'] == '行业监管':
        if industry in policy['targets']:
            impact_direction = '负面'
            impact_level = '高'
    
    return impact_direction, impact_level
```

### 4.3 政策风险识别

#### 常见政策风险
- **政策退出**：支持政策取消或减弱
- **监管加强**：合规成本增加
- **政策转向**：从支持变为限制
- **国际政策**：贸易壁垒、制裁

#### 风险评估指标
- **政策依赖度**：公司收入对政策的依赖程度
- **政策可持续性**：政策能否长期执行
- **政策不确定性**：政策变化的可能性
