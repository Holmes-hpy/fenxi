# 曼昆经济学原理深度学习总结（图表增强版）

## 一、经济学的十大原理

### 1.1 人们如何做出决策

#### 原理一：人们面临权衡取舍
```
      效率
        ↑
        │
        ● 平衡点
        │
        ↓
      平等
```
- 效率：社会能从其稀缺资源中得到最大利益
- 平等：这些利益在社会成员中平均分配
- 社会面临效率与平等的权衡

#### 原理二：机会成本
```
选择A ──→ 放弃B的价值 = 机会成本
    │
    ↓
  实际收益 - 机会成本 = 经济利润
```

#### 原理三：理性人考虑边际量
```
边际收益(MB)
    │
    ├── MB > MC → 增加活动
    │
    ├── MB = MC → 最优决策
    │
    └── MB < MC → 减少活动
    ↓
边际成本(MC)
```

### 1.2 人们如何相互影响

#### 原理六：看不见的手
```
      价格信号
         │
    ┌────┴────┐
    ↓         ↓
  生产者    消费者
    │         │
    └────┬────┘
         ↓
      市场均衡
```

#### 原理七：政府干预
```
市场失灵 ──→ 政府干预
     │
     ├── 外部性 → 税收/补贴
     │
     └── 市场势力 → 反垄断
```

### 1.3 整体经济如何运行

#### 原理十：菲利普斯曲线
```
        通货膨胀率
            ↑
            │
  短期PC    │  ┌─────────────────┐
            │  │                 │
            │  │      长期PC      │
            │  │      (垂直)      │
            │  └─────────────────┘
            └──────────────────────→ 失业率
                      自然失业率
```

## 二、供给与需求：市场如何运行

### 2.1 需求曲线
```
        P
        ↑
        │
        │       D
        │      /
        │     /
        │    /
        └───┴──→ Q
```

**需求定理**：价格↑ → 需求量↓

#### 需求曲线移动因素
| 因素 | 移动方向 | 图例 |
|------|---------|------|
| 收入增加（正常品） | 右移 | D→D' |
| 替代品价格上升 | 右移 | D→D' |
| 偏好改变 | 右移/左移 | D→D' |

### 2.2 供给曲线
```
        P
        ↑
        │
        │  S
        │    \
        │     \
        │      \
        └───┴──→ Q
```

**供给定理**：价格↑ → 供给量↑

#### 供给曲线移动因素
| 因素 | 移动方向 | 图例 |
|------|---------|------|
| 投入品价格下降 | 右移 | S→S' |
| 技术进步 | 右移 | S→S' |
| 预期价格上升 | 左移 | S→S' |

### 2.3 市场均衡
```
        P
        ↑
        │
   S    │       /
        │      /   \  D
均衡点  │     /   ●  \
        │    /     \  \
        │   /       \  \
        └──┴─────────┴──→ Q
            均衡数量
```

#### 均衡变动分析步骤
1. 确定事件影响供给还是需求
2. 确定曲线移动方向
3. 分析价格和数量变化

## 三、弹性及其应用

### 3.1 需求价格弹性
```
需求曲线形状与弹性
  P
  ↑
  │          缺乏弹性          单位弹性          富有弹性
  │    ┌──────┐         ┌──────────┐       ┌──────┐
  │   /       \       /           \     /       \
  │  /         \     /             \   /         \
  │ /           \   /               \ /           \
  └────────────────────────────────────────────────→ Q
       steep       moderate       flat
```

#### 弹性计算
```
Ed = (%ΔQ) / (%ΔP)
```

| 弹性类型 | Ed值 | 需求曲线 | 价格变动影响 |
|---------|------|---------|-------------|
| 富有弹性 | >1 | 平缓 | 价格↑→TR↓ |
| 缺乏弹性 | <1 | 陡峭 | 价格↑→TR↑ |
| 单位弹性 | =1 | 中等 | TR不变 |

### 3.2 需求收入弹性
```
Ei = (%ΔQ) / (%ΔI)

正常品：Ei > 0
高档品：Ei > 1
必需品：0 < Ei < 1
低档品：Ei < 0
```

### 3.3 交叉价格弹性
```
Exy = (%ΔQx) / (%ΔPy)

替代品：Exy > 0
互补品：Exy < 0
无关品：Exy = 0
```

## 四、消费者、生产者与市场效率

### 4.1 消费者剩余
```
        P
        ↑
        │     CS = 消费者剩余
        │    ┌─────────────┐
        │   /             \
        │  /               \
        │ /                 \
        │/                   \
        └─────────────────────→ Q
```
**消费者剩余 = 支付意愿 - 实际价格**

### 4.2 生产者剩余
```
        P
        ↑
        │                   PS = 生产者剩余
        │    ┌───────────────────────────┐
        │   /                           /
        │  /                           /
        │ /                           /
        │/                           /
        └─────────────────────────────→ Q
```
**生产者剩余 = 实际价格 - 成本**

### 4.3 总剩余
```
        P
        ↑
        │
        │     ▲ 无谓损失
        │    / \
        │   /   \
        │  /     \
        │ /       \
        └──────────→ Q
          Q最优   Q市场
```
**总剩余 = 消费者剩余 + 生产者剩余**

## 七、外部性

### 7.1 负外部性（污染）
```
        成本
        ↑
        │
MSC    │  /
        │ /
        │/
MPC    │
        └──────────→ 产量
            Q社会   Q市场
```
**社会最优 < 市场均衡**

### 7.2 正外部性（教育）
```
        收益
        ↑
        │
MSB    │
        │
MPB    │
        │    Q市场   Q社会
        └──────────→ 产量
```
**社会最优 > 市场均衡**

### 7.3 解决方案
| 方法 | 适用情况 | 效果 |
|------|---------|------|
| 矫正税 | 负外部性 | 使私人成本=社会成本 |
| 矫正补贴 | 正外部性 | 使私人收益=社会收益 |
| 可交易许可证 | 污染 | 灵活高效 |

## 十二、宏观经济数据

### 12.1 GDP组成
```
Y = C + I + G + NX
    │   │   │    │
    ▼   ▼   ▼    ▼
   消费 投资 政府 净出口
```

### 12.2 GDP核算方法
```
┌─────────────────────────────────┐
│        支出法                   │
│  C + I + G + NX                │
└────────────┬────────────────────┘
             │
             ▼
┌─────────────────────────────────┐
│        收入法                   │
│  工资 + 利息 + 租金 + 利润      │
└─────────────────────────────────┘
```

### 12.3 价格指数
```
GDP平减指数 = (名义GDP / 实际GDP) × 100

CPI = (一篮子物品价格 / 基年价格) × 100
```

## 十三、货币与通货膨胀

### 13.1 货币数量论
```
M × V = P × Y

M：货币供给
V：货币流通速度
P：物价水平
Y：实际GDP
```

### 13.2 货币与物价
```
        物价水平
            ↑
            │
            │  ───────────  货币供给↑
            │
            │  ───────────  货币供给不变
            │
            │  ───────────  货币供给↓
            └────────────────→ 时间
```

### 13.3 通货膨胀与利率
```
名义利率 = 实际利率 + 预期通货膨胀率

费雪效应：名义利率随通货膨胀率变动
```

## 十四、开放经济

### 14.1 汇率
```
        名义汇率
            ↑
            │
   升值     │  ──────→ 本国货币更值钱
            │
   贬值     │  ←────── 本国货币更便宜
            └────────────────→ 时间
```

### 14.2 购买力平价
```
一价定律：同一种物品在所有地方价格相同

实际汇率 = 名义汇率 × (国内物价 / 外国物价)
```

### 14.3 资本流动
```
净资本流出(NCO) = 本国购买外国资产 - 外国购买本国资产

NCO = -NX
```
