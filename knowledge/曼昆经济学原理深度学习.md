# 曼昆经济学原理深度学习总结

## 一、经济学的十大原理

### 1.1 人们如何做出决策

#### 原理一：人们面临权衡取舍
- "天下没有免费的午餐"
- 个人、社会、政府都面临权衡取舍
- 效率与平等的权衡最为重要
- **效率**：社会能从其稀缺资源中得到最大利益
- **平等**：这些利益在社会成员中平均分配

#### 原理二：某种东西的成本是为了得到它而放弃的东西
- **机会成本**：为了得到某种东西所必须放弃的东西
- 理性的决策者比较边际收益与边际成本
- **边际变动**：对行动计划的微小调整

#### 原理三：理性人考虑边际量
- 理性人：通过比较边际收益与边际成本来决策
- 边际收益 > 边际成本 → 增加活动
- 边际收益 < 边际成本 → 减少活动
- "Rational people think at the margin"

#### 原理四：人们会对激励做出反应
- 激励：引起人们采取某种行动的刺激
- 政策影响行为时需要考虑激励效应
- 市场中的价格激励
- 政策制定者需要考虑激励效应

### 1.2 人们如何相互影响

#### 原理五：贸易能使每个人状况更好
- 竞争使贸易成为零和博弈的观念是错误的
- 贸易使各国可以专门从事自己最擅长的活动
- 通过贸易，人们可以以更低的价格获得各种物品和服务
- 竞争带来双赢

#### 原理六：市场通常是组织经济活动的好方法
- **市场经济**：许多企业和家庭在物品与服务市场上相互交易
- 价格是引导决策的信号
- 价格反映商品的社会价值和社会成本
- **亚当·斯密的"看不见的手"**
  - 个人的利己行为在价格机制引导下促进社会利益
  - 价格调整使整个经济达到均衡

#### 原理七：政府有时可以改善市场结果
- 政府干预的原因：
  1. **促进效率**：解决市场失灵（外部性、市场势力）
  2. **促进平等**：影响经济福利的分配
- **市场失灵的两种情况**：
  1. **外部性**：一个人的行为对旁观者福利的影响
  2. **市场势力**：单个买者或卖者影响价格的能力

### 1.3 整体经济如何运行

#### 原理八：一国的生活水平取决于它生产物品与服务的能力
- 几乎所有生活水平的差异都可以归因于生产率的差异
- **生产率**：每单位劳动投入所生产的物品与服务数量
- 影响生产率的因素：
  - 物质资本
  - 人力资本
  - 自然资源
  - 技术知识

#### 原理九：当政府发行了过多货币时物价上升
- **通货膨胀**：经济中物价总水平的上升
- 物价上涨的原因：货币发行量过多
- 货币量与物价的关系：
  - 货币量↑ → 物价↑
  - 货币量↓ → 物价↓
- 货币流通速度稳定时，货币量决定物价

#### 原理十：社会面临通货膨胀与失业之间的短期权衡取舍
- 货币发行量增加 → 短期失业率降低
- 菲利普斯曲线：
  - 短期：通货膨胀与失业负相关
  - 长期：垂直的菲利普斯曲线
- 长期中不存在失业与通货膨胀的权衡

## 二、供给与需求：市场如何运行

### 2.1 市场的力量

#### 市场的定义
- 市场：由某种物品或服务的买者与卖者组成的一个群体
- 买者决定需求
- 卖者决定供给

#### 竞争市场
- 竞争市场：有许多买者和卖者，以至于每个人对价格的影响都微乎其微
- 完全竞争市场的特征：
  - 可供销售的物品完全相同
  - 买者和卖者都太多以至于没有人能影响价格

### 2.2 需求

#### 需求定理
- **需求定理**：在其他条件不变时，一种物品的价格上升，对该物品的需求量减少
- 需求量：买者愿意而且能够购买的一种物品的数量
- 需求表：表示某种物品价格与需求量之间关系的表格
- 需求曲线：表示某种物品价格与需求量之间关系的图形

#### 市场需求与个人需求
- 市场需求：所有个人对某种物品或服务的需求的总和
- 从个人需求到市场需求：将所有个人需求曲线在每个价格水平上横向加总

#### 需求曲线的移动
- **需求变动**：在每一个价格水平上需求量都发生改变
  - 需求曲线位置移动
- **需求量变动**：仅仅在价格变动时发生
  - 沿着需求曲线移动

#### 需求曲线移动的因素
1. **收入**：正常品↑ → 需求↑；低档品↑ → 需求↓
2. **相关物品价格**：
   - 替代品：一种物品价格↑ → 另一种需求↑
   - 互补品：一种物品价格↑ → 另一种需求↓
3. ** tastes（爱好）**
4. **预期**：对未来价格、收入的预期
5. **买者数量**：消费者数量↑ → 市场需求↑

### 2.3 供给

#### 供给定理
- **供给定理**：在其他条件不变时，一种物品价格上升，该物品供给量增加
- 供给量：卖者愿意而且能够出售的一种物品的数量
- 供给表：表示某种物品价格与供给量之间关系的表格
- 供给曲线：表示某种物品价格与供给量之间关系的图形

#### 市场供给
- 市场供给：所有卖者供给的总和
- 从个人供给到市场供给：横向加总

#### 供给曲线的移动
- **供给变动**：在每一个价格水平上供给量都发生改变
  - 供给曲线位置移动
- **供给量变动**：仅仅在价格变动时发生
  - 沿着供给曲线移动

#### 供给曲线移动的因素
1. **投入品价格**：投入品价格↑ → 供给↓
2. **技术**：技术进步 → 成本↓ → 供给↑
3. **预期**：预期价格↑ → 当前供给↓
4. **卖者数量**：卖者数量↑ → 市场供给↑

### 2.4 供给与需求的结合

#### 均衡
- **均衡**：供给与需求达到平衡的状态
- **均衡价格**：需求量等于供给量的价格
- **均衡数量**：均衡价格水平下的交易数量
- **超额供给**：供给量 > 需求量 → 价格下降
- **超额需求**：需求量 > 供给量 → 价格上涨

#### 分析均衡变动的步骤
1. 确定事件是移动供给曲线还是需求曲线
2. 确定移动方向（左还是右）
3. 用供求图说明均衡如何移动
4. 比较新均衡与原均衡：价格和产量如何变化

## 三、弹性及其应用

### 3.1 需求弹性

#### 需求价格弹性
- **需求价格弹性**：衡量需求量对价格变动的敏感程度
- 公式：Ed = (ΔQ/Q) / (ΔP/P)
- 富有弹性：Ed > 1，需求量对价格变动敏感
- 缺乏弹性：Ed < 1，需求量对价格变动不敏感
- 单位弹性：Ed = 1

#### 弹性与总收益
- 总收益（TR）= P × Q
- 富有弹性：价格↑ → 总收益↓
- 缺乏弹性：价格↑ → 总收益↑
- 单位弹性：价格变动不影响总收益

#### 影响需求弹性的因素
1. **必需品 vs 奢侈品**
   - 必需品：缺乏弹性
   - 奢侈品：富有弹性
2. **相近替代品的可获得性**
   - 有许多替代品：富有弹性
   - 几乎没有替代品：缺乏弹性
3. **市场的定义**
   - 狭窄定义：富有弹性
   - 宽泛定义：缺乏弹性
4. **时间范围**
   - 短期：缺乏弹性
   - 长期：富有弹性

#### 其他需求弹性
- **需求收入弹性**：Ed = (ΔQ/Q) / (ΔI/I)
  - 正常品：Ed > 0
  - 低档品：Ed < 0
- **需求的交叉价格弹性**：Ed = (ΔQa/Qa) / (ΔPb/Pb)
  - 替代品：正相关
  - 互补品：负相关

### 3.2 供给弹性

#### 供给价格弹性
- **供给价格弹性**：衡量供给量对价格变动的敏感程度
- 公式：Es = (ΔQ/Q) / (ΔP/P)
- 富有弹性：Es > 1
- 缺乏弹性：Es < 1
- 单位弹性：Es = 1

#### 影响供给弹性的因素
1. **卖者改变产量的灵活性**
   - 灵活：富有弹性
   - 不灵活：缺乏弹性
2. **时间范围**
   - 短期：缺乏弹性
   - 长期：富有弹性
3. **存储成本**
   - 存储成本低：富有弹性
   - 存储成本高：缺乏弹性

## 四、消费者、生产者与市场效率

### 4.1 消费者剩余

#### 支付意愿
- **支付意愿**：买者愿意为某种物品支付的最高价格
- 消费者剩余：买者愿意支付的最高价格 - 实际支付价格

#### 计算消费者剩余
- 消费者剩余 = 买者的支付意愿 - 实际支付价格
- 用需求曲线衡量消费者剩余
- 需求曲线以下、价格线以上的面积 = 消费者剩余

#### 价格变化对消费者剩余的影响
- 价格下降 → 消费者剩余增加
- 价格上升 → 消费者剩余减少

### 4.2 生产者剩余

#### 成本与销售意愿
- **成本**：卖者生产一种物品的机会成本
- 销售意愿：卖者愿意出售的最低价格

#### 计算生产者剩余
- 生产者剩余 = 实际获得价格 - 卖者的成本
- 用供给曲线衡量生产者剩余
- 价格线以下、供给曲线以上的面积 = 生产者剩余

### 4.3 市场效率

#### 消费者剩余 + 生产者剩余 = 总剩余
- **总剩余**：消费者剩余 + 生产者剩余
- = (买者的支付意愿 - 卖者的成本) 的总和

#### 自由市场的效率
- 自由市场配置资源的目标：使总剩余最大化
- 市场均衡使总剩余最大化
- 效率的三个条件：
  1. 边际买者的支付意愿 ≥ 边际卖者的成本
  2. 只生产边际买者想要的数量
  3. 在边际上，社会从交易中得到的利益 = 卖者承担的成本

#### 市场的局限性
- 市场失灵的原因：
  1. 外部性
  2. 市场势力（垄断）
- 外部性导致市场无效率
- 市场势力导致价格与边际成本分离

## 五、赋税的代价

### 5.1 赋税的无谓损失

#### 税收楔子
- 税收使买者支付的价格高于卖者得到的价格
- 税收在买者支付的价格和卖者得到的价格之间打入了一个楔子

#### 无谓损失
- **无谓损失**：税收导致的总剩余减少
- 无谓损失 = 税收收入减少 - 消费者剩余减少 - 生产者剩余减少
- 税收扭曲了激励，造成无谓损失

#### 无谓损失的决定因素
- 供给和需求的弹性越大，无谓损失越大
- 税收越大，无谓损失越大
- 无谓损失增加的速度比税收增加的速度快

### 5.2 税收与效率

#### 弹性与无谓损失
- 供给弹性↑ → 无谓损失↑
- 需求弹性↑ → 无谓损失↑

#### 税收的长期与短期
- 短期：供给和需求都缺乏弹性
- 长期：供给和需求都富有弹性
- 结论：长期税收的无谓损失 > 短期税收的无谓损失

### 5.3 税收的收入效应与替代效应

#### 替代效应
- 税收改变了行为的相对价格
- 促使人们转向被征税较少的替代活动

#### 收入效应
- 税收降低了人们的购买力
- 促使人们更加努力工作或减少消费

## 六、国际贸易

### 6.1 贸易的决定因素

#### 比较优势
- **绝对优势**：用比另一个生产者更少的投入生产某种物品
- **比较优势**：一个生产者以低于另一个生产者的机会成本生产某种物品
- 专业化与贸易使所有人受益
- 贸易的好处基于比较优势，而非绝对优势

#### 机会成本与贸易
- 如果一国某种物品的机会成本低于另一国，该国在这种物品上具有比较优势
- 各国应该专业化生产自己具有比较优势的物品
- 通过贸易进口自己处于比较劣势的物品

### 6.2 贸易的影响

#### 世界价格
- **世界价格**：一种物品在国际市场上通行的价格
- 比较国内价格与世界价格：
  - 国内价格 < 世界价格：该国具有比较优势
  - 国内价格 > 世界价格：该国处于比较劣势

#### 出口国的效应
- 出口使国内价格上升到世界价格
- 出口增加生产者剩余
- 出口减少消费者剩余
- 生产者剩余的增加 > 消费者剩余的减少
- 贸易增加一国的总剩余

#### 进口国的效应
- 进口使国内价格下降到世界价格
- 进口增加消费者剩余
- 进口减少生产者剩余
- 消费者剩余的增加 > 生产者剩余的减少
- 贸易增加一国的总剩余

### 6.3 各种限制贸易的观点

#### 关税
- **关税**：对进口物品征收的税
- 关税减少进口量
- 关税增加国内生产者剩余
- 关税减少消费者剩余
- 关税增加政府收入
- 关税造成无谓损失

#### 进口配额
- **进口配额**：对进口物品的数量限制
- 效果类似于关税
- 区别：政府不获得收入，外国卖者获得利润

#### 限制贸易的论点
1. **工作岗位论**：国际贸易导致某些工作机会丧失
2. **国家安全论**：某些行业对国家安全至关重要
3. **幼稚产业论**：新兴产业需要保护才能发展
4. **不公平竞争论**：他国政府补贴使竞争不公平
5. **作为讨价还价筹码论**：威胁征收关税以获取贸易让步

## 七、外部性

### 7.1 外部性的类型

#### 负外部性
- **负外部性**：一个人的行为对旁观者造成的不利影响
- 例子：污染、噪音
- 社会成本 > 私人成本
- 造成市场无效率

#### 正外部性
- **正外部性**：一个人的行为对旁观者的有利影响
- 例子：教育、疫苗接种
- 社会利益 > 私人利益
- 造成市场无效率

### 7.2 外部性与市场无效率

#### 负外部性与无效率
- 私人边际成本（MPC）
- 社会边际成本（MSC）= MPC + 外部成本
- 市场均衡产量 > 社会最优产量
- 无谓损失：过度生产

#### 正外部性与无效率
- 私人边际收益（MPB）
- 社会边际收益（MSB）= MPB + 外部收益
- 市场均衡产量 < 社会最优产量
- 无谓损失：生产不足

### 7.3 政府对策

#### 命令与控制政策
- 法规：禁止某些行为
- 标准：规定允许的污染水平
- 执行成本高，缺乏灵活性

#### 以市场为基础的政策
##### 矫正税
- **矫正税**：旨在纠正负外部性影响的税收
- 税额 = 外部成本
- 使私人决策考虑社会成本
- 激励企业减少污染
- 灵活性高，比命令控制政策更有效

##### 矫正补贴
- 用于正外部性
- 等于外部收益
- 鼓励更多生产

##### 可交易的污染许可证
- 政府发放污染许可证
- 企业可以买卖许可证
- 灵活性高，效率高

## 八、公共物品与公共资源

### 8.1 物品的分类

#### 竞争性与排他性
- **竞争性**：一个人使用一种物品减少其他人对该物品的使用
- **排他性**：可以阻止其他人使用一种物品

#### 四种类型

| 类型 | 竞争性 | 排他性 | 例子 |
|------|--------|--------|------|
| **私人物品** | ✓ | ✓ | 食物、衣服 |
| **公共物品** | ✗ | ✗ | 国防、灯塔 |
| **公共资源** | ✓ | ✗ | 海洋鱼类、公共牧场 |
| **自然垄断** | ✗ | ✓ | 消防、有线电视 |

### 8.2 公共物品

#### 公共物品的特点
- **非竞争性**：一个人使用不减少其他人使用
- **非排他性**：不能阻止其他人使用
- 例子：国防、灯塔、基础研究

#### 公共物品的困难
- **搭便车问题**：不支付费用就想获得利益
- 市场通常不能有效提供公共物品
- 需要政府介入

#### 公共物品举例
- **国防**：最重要的公共物品
- **基础研究**：知识是公共物品
- **反贫困**：解决贫困的社会保障计划

### 8.3 公共资源

#### 公共资源的特点
- **竞争性**：一个人使用减少其他人使用
- **非排他性**：不能阻止其他人使用
- 例子：海洋鱼类、公共牧场

#### 公共资源的困难
- **公地悲剧**：过度使用公共资源
- 私人决策者没有考虑外部成本
- 需要政府管制或税收

#### 公共资源举例
- **清洁的空气和水**：受到过度污染
- **拥堵的道路**：过度使用
- **野生动物**：过度捕猎

## 九、成本理论

### 9.1 机会成本

#### 机会成本的定义
- 机会成本：为得到某种东西而必须放弃的东西
- 会计成本 vs 经济成本
- 隐性成本 vs 显性成本

#### 理性人应该考虑所有机会成本
- 显性成本：直接货币支付
- 隐性成本：放弃的机会价值
- 正常利润：隐性成本的一部分

### 9.2 生产与成本

#### 生产函数
- **生产函数**：投入与产出之间的关系
- **边际产量**：增加一单位投入所增加的产量
- 边际产量递减：随着投入增加，边际产量减少

#### 总成本曲线
- **总成本（TC）** = 固定成本（FC） + 可变成本（VC）
- 固定成本：不随产量变化
- 可变成本：随产量变化

### 9.3 各种成本概念

#### 边际成本
- **边际成本（MC）**：增加一单位产量所增加的总成本
- MC = ΔTC/ΔQ
- 边际成本曲线是U形的
- 边际成本先下降后上升

#### 平均成本
- **平均固定成本（AFC）** = FC/Q
- **平均可变成本（AVC）** = VC/Q
- **平均总成本（ATC）** = TC/Q = AFC + AVC
- ATC曲线也是U形的
- 边际成本曲线与平均总成本曲线相交于ATC最低点

#### 长期与短期成本
- 短期：至少一种投入固定
- 长期：所有投入都可变
- 长期平均总成本曲线是短期平均总成本曲线的包络线
- **规模经济**：长期平均总成本随产量增加而下降
- **规模不经济**：长期平均总成本随产量增加而上升
- **规模收益不变**：长期平均总成本不随产量变化

## 十、竞争市场上的企业

### 10.1 竞争市场

#### 竞争市场的特点
- 有许多买者和卖者
- 所有企业销售的产品基本相同
- 企业可以自由进入或退出市场
- 每个企业都是价格接受者

### 10.2 竞争企业的收益

#### 收益的基本概念
- **总收益（TR）** = P × Q
- **平均收益（AR）** = TR/Q = P
- **边际收益（MR）** = ΔTR/ΔQ

#### 竞争企业的特点
- 价格 = 平均收益 = 边际收益
- P = AR = MR

### 10.3 利润最大化

#### 产量选择
- 理性企业选择使利润最大化的产量
- 边际收益 = 边际成本时，利润最大
- MR = MC 原则适用于所有企业

#### 企业的短期决策
- 价格 > 平均可变成本：继续经营
- 价格 < 平均可变成本：停止经营
- 关闭点：P = AVC的最低点

#### 企业的长期决策
- 如果TR > TC：进入市场
- 如果TR < TC：退出市场
- 长期均衡：在P = ATC最低点经营

### 10.4 竞争市场的供给曲线

#### 短期供给曲线
- 竞争企业的短期供给曲线：MC曲线在AVC以上的部分
- 市场供给曲线：所有企业供给的横向加总

#### 长期供给曲线
- 企业进入和退出使经济利润趋近于零
- 长期均衡：P = ATC最低点 = MC
- 长期供给曲线：在LAC最低点水平的水平线

## 十一、垄断

### 11.1 垄断的原因

#### 垄断的定义
- **垄断**：单个企业是某种产品的唯一卖者
- 没有相近的替代品

#### 垄断的三个来源
1. **关键资源由一家企业拥有**
   - 例子：钻石De Beers
2. **政府给予一个企业排他性权利**
   - 例子：专利、版权
3. **生产成本使一个企业比多个企业更有效率**
   - 自然垄断：一个企业能以低于两个或更多企业的成本供给整个市场

### 11.2 垄断企业的行为

#### 垄断企业的产量决策
- 垄断企业面临整个市场的需求曲线
- 垄断企业的MR ≠ P
- MR < P（需求曲线下方）
- MR曲线在需求曲线下方
- 垄断企业选择使MR = MC的产量

#### 垄断企业的定价
- 根据需求曲线确定价格
- 垄断企业是价格制定者
- 产量 < 竞争产量
- 价格 > 竞争价格

#### 垄断企业的无效率
- 垄断产量 < 社会最优产量
- 垄断价格 > 边际成本
- 造成无谓损失
- 垄断企业没有竞争压力，效率低下

### 11.3 价格歧视

#### 价格歧视的定义
- **价格歧视**：以不同价格向不同顾客出售同一种物品的做法

#### 价格歧视的例子
- 电影票、飞机票、优惠券

#### 价格歧视的逻辑
- 企业向不同顾客收取不同价格
- 顾客的支付意愿不同
- 价格歧视可以增加垄断企业的利润

#### 价格歧视与福利
- 完全价格歧视：垄断企业知道每个顾客的支付意愿
- 完全价格歧视下，无谓损失消失
- 社会总剩余等于完全竞争市场的水平

### 11.4 对垄断的公共政策

#### 反托拉斯法
- 《谢尔曼反托拉斯法》（1890）
- 《克莱顿反托拉斯法》（1914）

#### 管制
- 政府可以管制垄断企业的定价
- 自然垄断的价格管制：P = ATC
- 边际成本定价：P = MC

#### 公有制
- 政府所有制
- 例子：公用事业

#### 不作为
- 有时政府不干预垄断
- 理由：垄断利润可能刺激创新

## 十二、寡头

### 12.1 寡头市场

#### 寡头的定义
- **寡头**：只有少数几个卖者的市场
- 每个企业的行动都影响其他企业

#### 寡头的特征
- 只有少数几家企业
- 产品可以是同质的或有差异的
- 进入壁垒高
- 企业之间相互依赖

### 12.2 寡头之间的竞争

#### 博弈论
- **博弈论**：研究策略互动情况下的决策
- **策略**：一个人在考虑其他人反应的情况下所作的决定

#### 寡头的均衡
- **纳什均衡**：相互作用的企业中，每个企业都选择了对自己最有利的策略
- **囚徒困境**：个人理性与集体理性的冲突

#### 寡头的囚徒困境
- 如果两家企业合作，将获得垄断利润
- 但每家企业都有作弊的激励
- 结果：两家企业都只获得竞争利润

### 12.3 寡头市场的均衡

#### 竞争与垄断之间的市场
- 寡头市场的结果：
  - 竞争结果：低价格、高产量
  - 垄断结果：高价格、低产量
- 寡头数量越多 → 越接近竞争结果
- 寡头数量越少 → 越接近垄断结果

#### 卡特尔
- **卡特尔**：企业之间关于产量和定价的协议
- OPEC是著名的卡特尔
- 卡特尔面临囚徒困境
- 协议往往难以维持

### 12.4 寡头与公共政策

#### 反对寡头的观点
- 价格高于边际成本
- 产量低于社会最优
- 垄断利润是社会损失

#### 支持寡头的观点
- 大企业可以实现规模经济
- 大企业有更强的创新能力
- 寡头之间的竞争仍然存在

## 十三、薪酬理论

### 13.1 劳动市场

#### 边际产品收益
- **边际产品收益（MRP）**：增加一单位劳动所增加的收益
- MRP = MPL × P
- 企业雇用劳动直到：MRP = 工资

#### 劳动的需求
- 劳动需求取决于：
  - 劳动的边际产品
  - 产品的价格
- 劳动需求曲线向右下方倾斜

#### 劳动的供给
- 劳动供给取决于：
  - 工资
  - 工作与非工作的权衡
- 劳动供给曲线向右上方倾斜

### 13.2 劳动市场的均衡

#### 均衡工资
- 均衡工资 = 劳动供给 = 劳动需求
- 工资调整使劳动市场出清

#### 劳动市场均衡的决定因素
- 生产效率
- 劳动市场的供给和需求

### 13.3 工资差异

#### 为什么工资不同？
1. **补偿性工资差异**：补偿非货币特质
2. **劳动的效率差异**：能力差异
3. **教育**：人力资本投资
4. **经验**：在职培训
5. **歧视**：对某些群体的偏见

#### 人力资本
- **人力资本**：个人通过教育、培训和经验积累的知识和技能
- 教育投资：
  - 成本：直接成本 + 机会成本
  - 收益：未来更高的工资

#### 超级明星
- **超级明星现象**：在某些行业，最优秀的人得到不成比例的高收入
- 条件：
  1. 消费者愿意为最优秀者支付高价格
  2. 供给受限于技术

## 十四、宏观经济数据

### 14.1 国内生产总值

#### GDP的定义
- **GDP（国内生产总值）**：一定时期内一国境内生产的所有最终物品与服务的市场价值

#### GDP的组成部分
- **Y = C + I + G + NX**
- C：消费
- I：投资
- G：政府购买
- NX：净出口

#### GDP的核算方法
1. **支出法**：加总所有支出
2. **收入法**：加总所有收入

### 14.2 价格指数

#### 通货膨胀率
- **GDP平减指数** = (名义GDP / 实际GDP) × 100
- 通货膨胀率 = (今年GDP平减指数 - 去年GDP平减指数) / 去年GDP平减指数 × 100%

#### 消费者物价指数（CPI）
- CPI = (一篮子物品和服务的价格 / 基年价格) × 100
- CPI衡量典型消费者购买的一篮子物品
- CPI是更常用的通货膨胀指标

### 14.3 失业率

#### 失业的定义
- **劳动力**：就业者 + 失业者
- **失业率** = 失业者 / 劳动力 × 100%

#### 自然失业率
- **自然失业率**：失业率围绕其波动的平均水平
- 自然失业率是经济的正常失业率

## 十五、货币与通货膨胀

### 15.1 货币

#### 货币的定义
- **货币**：经济中人们用于相互购买物品与服务的东西
- 货币的功能：
  1. 交换媒介
  2. 价值储藏
  3. 计价单位

#### 货币的类型
- **商品货币**：本身有价值的货币（黄金）
- **法定货币**：政府法令宣布作为货币（纸币）

### 15.2 货币数量论

#### 货币方程
- **M × V = P × Y**
  - M：货币供给
  - V：货币流通速度
  - P：物价水平
  - Y：实际GDP

#### 货币与物价的关系
- 货币流通速度（V）基本稳定
- 实际GDP（Y）由生产要素和生产函数决定
- 因此：P与M同比例变动
- **货币数量论**：物价水平的变化源于货币供给的变化

#### 货币与通货膨胀
- 货币供给↑ → 物价水平↑ → 通货膨胀
- 货币超发是通货膨胀的根本原因

### 15.3 货币铸造税

- **货币铸造税**：政府通过发行货币筹集的收入
- 当政府通过印刷货币筹资时：
  - 增加货币供给
  - 引起通货膨胀
  - 相当于向公众征税

## 十六、开放经济

### 16.1 资本与产品的流动

#### 国际资本流动
- **净资本流出（NCO）**：本国居民购买的外国资产 - 外国居民购买的本国资产
- NCO > 0：本国是资本净流出国
- NCO < 0：本国是资本净流入国

#### 国际贸易
- **贸易余额（NX）**：出口 - 进口
- NX = Y - (C + I + G)
- NX = -NCO（储蓄-投资恒等式）

### 16.2 汇率

#### 名义汇率
- **名义汇率**：一种货币兑换另一种货币的比率
- 升值：本国货币汇率上升
- 贬值：本国货币汇率下降

#### 实际汇率
- **实际汇率**：两国物品的相对价格
- 实际汇率 = 名义汇率 × (国内物价 / 外国物价)

### 16.3 购买力平价

#### 一价定律
- 同一种物品在所有地方价格相同
- 前提：没有运输成本，没有贸易壁垒

#### 购买力平价理论
- 长期：名义汇率由相对物价决定
- 货币贬值的国家物价水平上升
- 货币升值的国家物价水平下降

## 十七、失业与货币政策

### 17.1 自然失业率

#### 自然失业率的决定因素
- 离职率（l）
- 失业者找到工作的比率（f）
- 自然失业率 = l / (l + f)

#### 失业的类型
- **摩擦性失业**：工作转换期间的失业
- **结构性失业**：工资高于均衡工资

#### 效率工资
- **效率工资**：企业为了提高生产率而支付的高于均衡的工资
- 效率工资理论解释为什么工资高于均衡：
  1. 工人健康
  2. 工人素质
  3. 工人努力程度
  4. 降低人员流动

### 17.2 菲利普斯曲线

#### 菲利普斯曲线的定义
- **菲利普斯曲线**：通货膨胀与失业之间的负相关关系
- 短期：向右下方倾斜
- 长期：垂直的菲利普斯曲线

#### 菲利普斯曲线的移动
- 预期通货膨胀↑ → 菲利普斯曲线向右上方移动
- 供给冲击 → 菲利普斯曲线移动

#### 滞胀
- **滞胀**：高通货膨胀与高失业并存
- 供给冲击（石油价格）可能导致滞胀

## 学习心得

### 核心洞察

1. **经济学思维方式的培养**
   - 学会权衡取舍
   - 考虑边际量
   - 理解激励效应
   - 认识到激励与行为的关系

2. **市场机制的理解**
   - 供给与需求的相互作用
   - 价格是市场的核心信号
   - 亚当·斯密的"看不见的手"
   - 市场失灵需要政府干预

3. **效率与平等的权衡**
   - 效率：资源配置使总剩余最大化
   - 平等：利益在社会成员中平均分配
   - 两者往往存在冲突

4. **宏观经济的整体视角**
   - GDP反映经济整体表现
   - 货币供给决定物价水平
   - 失业与通货膨胀存在权衡
   - 开放经济中汇率的重要性

### 实践应用

1. **投资决策框架**
   ```
   基本面分析：
   ├→ 宏观经济环境（GDP、CPI、利率）
   ├→ 行业周期位置（供需分析）
   ├→ 公司竞争力（成本结构、盈利能力）
   └→ 市场结构（竞争程度、垄断程度）
   
   技术分析：
   └→ 价格走势、交易量分析
   ```

2. **风险管理**
   - 理解系统性风险与非系统性风险
   - 分散化投资降低风险
   - 对冲策略的应用
   - 期权等金融工具的风险管理

3. **政策分析**
   - 货币政策的影响：利率、货币供给
   - 财政政策的影响：税收、政府支出
   - 国际贸易政策的影响
   - 产业政策的影响

### 经济学原理在投资中的应用

1. **宏观分析**
   - 利用GDP、CPI、PMI等指标判断经济周期
   - 分析货币政策走向
   - 评估财政政策影响

2. **行业分析**
   - 利用供给需求原理分析行业前景
   - 评估行业弹性特征
   - 判断行业生命周期

3. **公司分析**
   - 利用成本理论分析公司竞争力
   - 理解不同市场结构下的公司行为
   - 评估公司定价能力

4. **国际市场分析**
   - 利用比较优势理论分析国际贸易
   - 分析汇率变动的影响
   - 评估国际资本流动
