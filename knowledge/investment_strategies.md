## 一、核心投资策略

### 1.1 价值投资策略

**核心理念**：
- 股票是企业的一部分
- 投资的是企业未来现金流
- 价格围绕价值波动，最终会回归价值

**操作要点**：

1. **选择优质公司**
   - 行业龙头
   - 竞争优势明显
   - 管理层诚信有能力
   - 财务数据健康

2. **估值合理或偏低**
   - PE/PB处于历史低位
   - 或相对行业折价
   - 有安全边际

3. **长期持有**
   - 不关注短期波动
   - 等待价值回归
   - 分享公司成长红利

**经典案例**：
- 巴菲特投资可口可乐
- 段永平投资贵州茅台
- 彼得·林奇投资沃尔玛

### 1.2 成长投资策略

**核心理念**：
- 投资高成长的公司
- 享受业绩增长带来的股价上涨
- 成长性是最大的安全边际

**操作要点**：

1. **选择成长行业**
   - 行业渗透率快速提升
   - 政策支持
   - 技术革命

2. **选择行业龙头**
   - 市占率提升
   - 技术领先
   - 规模优势

3. **适当分散**
   - 成长股波动大
   - 行业集中风险
   - 仓位控制

**关键指标**：
- 营收增速 > 30%
- 利润增速 > 30%
- 市值空间大

**风险提示**：
- 估值容易泡沫化
- 竞争激烈
- 技术路线变化

### 1.3 趋势投资策略

**核心理念**：
- 顺势而为
- 趋势是你的朋友
- 不要逆势而行

**操作要点**：

1. **识别趋势**
   - 均线多头排列
   - 成交量配合
   - 价格创新高

2. **跟随趋势**
   - 不抄底
   - 不预测顶
   - 趋势破坏离场

3. **严格止损**
   - 趋势破坏必须离场
   - 不要幻想反弹
   - 保住本金

**技术指标**：
- 均线系统（MA5/MA10/MA20）
- MACD指标
- 趋势线
- 成交量

### 1.4 量化投资策略

**核心理念**：
- 用数学和统计方法
- 建立量化模型
- 克服人性弱点

**主要方法**：

1. **多因子模型**
   - 价值因子：PE、PB、PS
   - 质量因子：ROE、毛利率
   - 动量因子：近期涨幅
   - 规模因子：市值大小

2. **统计套利**
   - 配对交易
   - 均值回归
   - 收敛交易

3. **机器学习**
   - 预测模型
   - 分类模型
   - 聚类分析

**优势**：
- 克服人性弱点
- 可回测验证
- 大规模执行

**劣势**：
- 模型风险
- 市场变化
- 执行成本

## 二、资产配置策略

### 2.1 大类资产配置

**核心框架**：
```
股票 + 债券 + 商品 + 现金
```

**配置原则**：
- 年龄法则：债券比例 = 年龄%
- 风险偏好：高风险偏好可增加股票比例
- 流动性需求：短期用钱不投高风险资产

**动态调整**：
- 高估值减股票
- 低估值加股票
- 债券作为稳定器

### 2.2 行业配置策略

**核心框架**：
```
核心资产（70%）+ 卫星资产（30%）
```

**核心资产**：
- 行业龙头
- 稳定增长
- 低波动
- 长持为主

**卫星资产**：
- 新兴行业
- 政策主题
- 高弹性
- 灵活调整

**行业轮动策略**：
- 复苏期：加金融、周期、消费
- 过热期：加商品、周期
- 滞胀期：加现金、防御
- 衰退期：加债券、防御

### 2.3 仓位管理

**核心原则**：
- 不满仓
- 不空仓
- 保持灵活性

**仓位控制**：

| 市场状态 | 仓位建议 | 说明 |
|---------|---------|------|
| 大牛市 | 80-100% | 享受泡沫 |
| 正常偏牛 | 60-80% | 积极进攻 |
| 震荡市 | 40-60% | 灵活应对 |
| 正常偏熊 | 20-40% | 谨慎防御 |
| 大熊市 | 0-20% | 保存实力 |

**金字塔建仓**：
```
价格低 → 仓位重
价格高 → 仓位轻
```

## 三、风险管理策略

### 3.1 止损策略

**核心原则**：
- 保住本金是第一位
- 及时止损，避免深度套牢
- 不要幻想反弹

**止损方法**：

| 方法 | 说明 | 适用场景 |
|-----|------|---------|
| 固定止损 | 亏损10%必须卖出 | 所有策略 |
| 技术止损 | 跌破重要均线卖出 | 趋势投资 |
| 逻辑止损 | 买入逻辑被证伪卖出 | 价值投资 |

**止损纪律**：
1. 买入前设定止损位
2. 亏损达到止损位立即卖出
3. 不要临时更改止损位
4. 记录每次止损原因

### 3.2 分散化策略

**核心原则**：
- 不要把所有鸡蛋放在一个篮子里
- 相关性低的资产组合效果更好
- 适度分散，不要过度分散

**分散化方法**：

1. **行业分散**
   - 不同行业组合
   - 避免行业集中风险

2. **风格分散**
   - 价值 + 成长
   - 大盘 + 小盘
   - 国内 + 海外

3. **资产分散**
   - 股票 + 债券
   - 股票 + 商品
   - 股票 + 现金

### 3.3 对冲策略

**核心思想**：
- 用相关资产对冲风险
- 降低组合波动
- 在不确定性中寻找确定性

**常用方法**：

1. **期货对冲**
   - 股指期货对冲系统性风险
   - 商品期货对冲原材料风险

2. **期权对冲**
   - 买入认沽期权保护
   - 卖出认购期权增收

3. **配对交易**
   - 做多强势股
   - 做空弱势股
   - 获取相对收益

## 四、交易策略

### 4.1 选股策略

**价值选股**：
```
PE < 行业平均
PB < 行业平均
ROE > 15%
净利润持续增长
现金流优秀
```

**成长选股**：
```
营收增速 > 30%
利润增速 > 30%
市占率持续提升
行业空间大
```

**技术选股**：
```
均线多头排列
成交量放大
MACD金叉
RSI适中（40-70）
```

### 4.2 买入策略

**定投策略**：
- 定期定额投资
- 无视市场波动
- 长期坚持

**分批建仓**：
```
第一次：30%（建立底仓）
第二次：30%（回调加仓）
第三次：40%（确认后重仓）
```

**金字塔买入**：
```
价格低 → 买得多
价格高 → 买得少
成本摊低效果好
```

### 4.3 卖出策略

**止盈策略**：
- 达到预期收益（20-30%）考虑减仓
- 估值出现泡沫时分批卖出
- 高位放巨量可能是见顶信号

**卖出信号**：
```
1. 估值显著高估
2. 行业景气度下降
3. 公司竞争优势减弱
4. 发现更好的机会
5. 系统性风险来临
```

**核心理念**：
- 北向资金是A股市场的重要增量资金
- 外资流向往往领先市场走势
- 外资重仓股的表现与北向资金高度相关

**操作要点**：

1. **每日监测**
   - 记录北向资金流向（沪股通+深股通）
   - 区分外资偏好板块（消费、金融）
   - 关注连续流出/流入趋势

2. **仓位调整规则**
   ```
   | 北向资金信号 | 仓位建议 | 操作 |
   |-------------|---------|------|
   | 单日流出>30亿 | 降至60% | 减仓外资重仓股 |
   | 连续流出>50亿 | 降至50% | 切换至内资主导板块 |
   | 连续流出>100亿 | 降至40% | 增加防御板块配置 |
   ```

   - 贵州茅台、宁德时代、招商银行等

**注意事项**：

### 7.2 AI算力产业链投资策略

**策略名称**："卖水人"策略

**核心理念**：
- 不管AI最终谁胜出，都要买基础设施
- 基础设施的业绩确定性最高
- 上游设备/材料比中游制造更抗风险

**产业链分析**：

```
AI算力产业链：
├── 上游：芯片（GPU、ASIC）  ❌ 风险高
│   └── 受制裁、技术壁垒高
│
├── 中游：服务器、存储、光模块  ✅ 机会大
│   └── 业绩确定性高、国产替代快
│
└── 下游：算力租赁、应用  ⚠️ 风险中
    └── 商业模式不确定、竞争激烈
```

**优先配置**：
1. **PCB板**（印制电路板）
   - 工业之母，AI服务器必备
   - 龙头：深南电路(002916)、沪电股份(002463)
   
2. **光模块**（光通信）
   - 高速光模块需求爆发
   - 龙头：中际旭创(300308)、新易盛(300502)

3. **高速连接器**
   - 算力服务器核心配件
   - 龙头：意华股份(002897)

4. **散热系统**
   - AI服务器功耗大，散热需求高
   - 龙头：英维克(002837)、高澜股份(300499)

**选股标准**：
- 必须是行业龙头
- 业绩增长确定（营收增速>30%）
- 估值合理（PE<60）
- 成交活跃（换手率>2%）

**风险提示**：
- 题材炒作波动大
- 估值可能泡沫化
- 需要设置止损位

### 7.3 行业轮动策略

**策略名称**：防御-进攻轮动策略

**核心理念**：
- 根据经济周期和市场环境，调整进攻/防御配置
- 复苏期加进攻，过热期减进攻
- 滞胀期加防御，衰退期重防御

**经济周期与行业配置**：

```
经济周期 → 行业轮动

衰退期（经济下行）：
├── 特征：GDP下行、通胀下行
├── 进攻：❌ 不宜
├── 防御：✅✅✅ 增加
│   ├── 债券
│   ├── 黄金
│   ├── 必需消费
│   └── 公用事业

复苏期（经济上行）：
├── 特征：GDP上行、通胀低位
├── 进攻：✅✅✅ 增加
│   ├── 消费
│   ├── 科技
│   ├── 周期
│   └── 金融
├── 防御：❌ 减少

过热期（经济过热）：
├── 特征：GDP高位、通胀上行
├── 进攻：✅✅ 增加
│   ├── 商品
│   ├── 原材料
│   └── 能源
├── 防御：❌ 减少

滞胀期（经济下行+通胀）：
├── 特征：GDP下行、通胀高位
├── 进攻：❌ 不宜
├── 防御：✅✅ 增加
│   ├── 煤炭
│   ├── 电力
│   ├── 银行
│   └── 现金
```

**当前市场判断**：

- 建议配置：煤炭(+3.93%)、电力(+1.47%)、银行

**操作建议**：
- 防御板块：50%

### 7.4 热点题材操作规范

**策略名称**：热点题材"三不"原则

**核心理念**：
- 热点题材是高风险高收益
- 没有止损纪律的题材炒作是赌博
- 分散持仓是降低风险的有效方法

**"三不"原则**：

1. **不追高**
   - 等回调5-10%再介入
   - 放量突破后等待回踩确认
   - 连续涨停后不追

2. **不止损不止**
   - 止损是铁律，必须执行
   - 热点题材止损位：-7%
   - 触及止损必须卖出

3. **不满仓**
   - 单只热点股仓位≤10%
   - 热点股总仓位≤20%
   - 保留足够现金

**买入前检查清单**：
- [ ] 业绩能支撑估值吗？
- [ ] 行业龙头还是纯炒作？
- [ ] 设置止损位了吗？
- [ ] 能承受多大亏损？
- [ ] 仓位是否过重？

**卖出决策**：
- [ ] 触及止损位？→ 必须卖出
- [ ] 达到目标收益（+20%）？→ 考虑减仓
- [ ] 发现更好机会？→ 权衡后决策
- [ ] 逻辑被证伪？→ 果断卖出

### 7.5 仓位管理优化方案

**策略名称**：动态仓位管理策略

**核心理念**：
- 仓位应根据市场环境动态调整
- 高不确定性时保持轻仓
- 低不确定性时可以重仓

**仓位决策矩阵**：

```
| 市场状态 | 仓位范围 | 配置策略 |
|---------|---------|---------|
| 大牛市 | 80-100% | 满仓进攻 |
| 正常偏牛 | 60-80% | 积极进攻 |
| 震荡市 | 40-60% | 均衡配置 |
| 正常偏熊 | 20-40% | 谨慎防御 |
| 大熊市 | 0-20% | 保存实力 |

市场状态判断标准：

牛市信号：
✅ 沪深300突破年线
✅ 北向资金持续流入
✅ 成交量放大
✅ 涨停家数>100

熊市信号：
❌ 沪深300跌破年线
❌ 北向资金持续流出
❌ 成交量萎缩
❌ 跌停家数>50
```

**动态调整规则**：

1. **每周评估**
   - 检查市场状态信号

### 7.6 选股策略验证体系

**策略名称**：策略有效性量化评估

**核心理念**：
- 选股策略需要长期验证
- 量化指标是评估策略有效性的标准
- 持续优化是提高胜率的关键

**量化评估指标**：

| 指标 | 计算方法 | 目标值 | 说明 |
|-----|---------|-------|------|
| 胜率 | 盈利次数/总次数 | >50% | 越高越好 |
| 盈亏比 | 平均盈利/平均亏损 | >1.5 | 至少1.5倍以上 |
| 最大回撤 | 策略最大亏损 | <20% | 越小越好 |
| 夏普比率 | (收益-无风险)/波动率 | >1.0 | 风险调整收益 |
| 卡玛比率 | 收益率/最大回撤 | >2.0 | 收益/风险比 |

**验证流程**：

```
1. 记录每次选股
   - 股票代码、名称
   - 选股日期
   - 选股理由
   - 策略评分

2. 跟踪持仓表现
   - 每日记录涨跌幅
   - 计算相对收益
   - 记录操作（买入/卖出/止损）

3. 周度评估
   - 统计本周选股胜率
   - 计算平均盈亏
   - 分析成败原因

4. 月度复盘
   - 统计本月选股表现
   - 评估策略有效性
   - 优化策略参数

5. 季度调整
   - 回顾季度表现
   - 评估策略稳定性
   - 决定是否继续使用
```

### 9.1 高低切换应对策略

**策略名称**：市场风格切换应对策略

**核心理念**：
- 市场风格切换是A股的常态
- 切换前必然出现"极致抱团"
- 切换中应该"先减仓高位，后加仓低位"
- 切换后可能反复，需要"渐进式调仓"

**高低切换的预警信号**：

```
1. 单一板块涨幅超50%且无回调
   - 例如：本轮AI算力4-5月累计涨幅>30%
   - 部分龙头如中际旭创等涨幅>50%

2. 估值分位超90%
   - 科创50 PE(TTM) 78倍（历史95%分位）
   - 部分光模块龙头PE>60倍

3. 资金面信号
   - 公募基金极致抱团
   - 北向资金开始调仓
   - ETF大额净流出

4. 政策面信号
   - 监管层出台"风格管控"政策
   - 公募新规限制"风格漂移"

5. 外部冲击
   - 美股科技股大幅调整
   - 美联储加息预期升温
```

**切换中的操作步骤**：

| 阶段 | 时间窗口 | 操作 | 仓位 |
|:----:|:-------:|:----:|:----:|
| 预警期 | 切换前1-2周 | 减仓高位股 30% | 高位股降至5% |
| 确认期 | 切换前1-3天 | 减仓高位股至3%以下 | 增配防御板块 |
| 切换期 | 切换中1-2周 | 完成调仓 | 防御+现金50% |
| 新主线期 | 切换后 | 增配新主线 | 核心+卫星 |

**新主线识别标准**：
1. **政策催化**：明确的产业政策
2. **资金流入**：北向+公募+ETF同步流入
3. **业绩验证**：一季报或中报业绩超预期
4. **估值合理**：PE分位<70%
