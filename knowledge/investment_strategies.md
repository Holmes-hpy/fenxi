## 一、核心投资策略

### 1.1 价值投资策略

**核心理念**：
- 股票是企业的一部分
- 投资的是企业未来现金流
- 价格围绕价值波动，最终会回归价值

**操作要点**：

1. **选择优质公司**
   - 行业龙头
   - 竞争优势明显
   - 管理层诚信有能力
   - 财务数据健康

2. **估值合理或偏低**
   - PE/PB处于历史低位
   - 或相对行业折价
   - 有安全边际

3. **长期持有**
   - 不关注短期波动
   - 等待价值回归
   - 分享公司成长红利

**经典案例**：
- 巴菲特投资可口可乐
- 段永平投资贵州茅台
- 彼得·林奇投资沃尔玛

### 1.2 成长投资策略

**核心理念**：
- 投资高成长的公司
- 享受业绩增长带来的股价上涨
- 成长性是最大的安全边际

**操作要点**：

1. **选择成长行业**
   - 行业渗透率快速提升
   - 政策支持
   - 技术革命

2. **选择行业龙头**
   - 市占率提升
   - 技术领先
   - 规模优势

3. **适当分散**
   - 成长股波动大
   - 行业集中风险
   - 仓位控制

**关键指标**：
- 营收增速 > 30%
- 利润增速 > 30%
- 市值空间大

**风险提示**：
- 估值容易泡沫化
- 竞争激烈
- 技术路线变化

### 1.3 趋势投资策略

**核心理念**：
- 顺势而为
- 趋势是你的朋友
- 不要逆势而行

**操作要点**：

1. **识别趋势**
   - 均线多头排列
   - 成交量配合
   - 价格创新高

2. **跟随趋势**
   - 不抄底
   - 不预测顶
   - 趋势破坏离场

3. **严格止损**
   - 趋势破坏必须离场
   - 不要幻想反弹
   - 保住本金

**技术指标**：
- 均线系统（MA5/MA10/MA20）
- MACD指标
- 趋势线
- 成交量

### 1.4 量化投资策略

**核心理念**：
- 用数学和统计方法
- 建立量化模型
- 克服人性弱点

**主要方法**：

1. **多因子模型**
   - 价值因子：PE、PB、PS
   - 质量因子：ROE、毛利率
   - 动量因子：近期涨幅
   - 规模因子：市值大小

2. **统计套利**
   - 配对交易
   - 均值回归
   - 收敛交易

3. **机器学习**
   - 预测模型
   - 分类模型
   - 聚类分析

**优势**：
- 克服人性弱点
- 可回测验证
- 大规模执行

**劣势**：
- 模型风险
- 市场变化
- 执行成本

## 二、资产配置策略

### 2.1 大类资产配置

**核心框架**：
```
股票 + 债券 + 商品 + 现金
```

**配置原则**：
- 年龄法则：债券比例 = 年龄%
- 风险偏好：高风险偏好可增加股票比例
- 流动性需求：短期用钱不投高风险资产

**动态调整**：
- 高估值减股票
- 低估值加股票
- 债券作为稳定器

### 2.2 行业配置策略

**核心框架**：
```
核心资产（70%）+ 卫星资产（30%）
```

**核心资产**：
- 行业龙头
- 稳定增长
- 低波动
- 长持为主

**卫星资产**：
- 新兴行业
- 政策主题
- 高弹性
- 灵活调整

**行业轮动策略**：
- 复苏期：加金融、周期、消费
- 过热期：加商品、周期
- 滞胀期：加现金、防御
- 衰退期：加债券、防御

### 2.3 仓位管理

**核心原则**：
- 不满仓
- 不空仓
- 保持灵活性

**仓位控制**：

| 市场状态 | 仓位建议 | 说明 |
|---------|---------|------|
| 大牛市 | 80-100% | 享受泡沫 |
| 正常偏牛 | 60-80% | 积极进攻 |
| 震荡市 | 40-60% | 灵活应对 |
| 正常偏熊 | 20-40% | 谨慎防御 |
| 大熊市 | 0-20% | 保存实力 |

**金字塔建仓**：
```
价格低 → 仓位重
价格高 → 仓位轻
```

## 三、风险管理策略

### 3.1 止损策略

**核心原则**：
- 保住本金是第一位
- 及时止损，避免深度套牢
- 不要幻想反弹

**止损方法**：

| 方法 | 说明 | 适用场景 |
|-----|------|---------|
| 固定止损 | 亏损10%必须卖出 | 所有策略 |
| 技术止损 | 跌破重要均线卖出 | 趋势投资 |
| 逻辑止损 | 买入逻辑被证伪卖出 | 价值投资 |

**止损纪律**：
1. 买入前设定止损位
2. 亏损达到止损位立即卖出
3. 不要临时更改止损位
4. 记录每次止损原因

### 3.2 分散化策略

**核心原则**：
- 不要把所有鸡蛋放在一个篮子里
- 相关性低的资产组合效果更好
- 适度分散，不要过度分散

**分散化方法**：

1. **行业分散**
   - 不同行业组合
   - 避免行业集中风险

2. **风格分散**
   - 价值 + 成长
   - 大盘 + 小盘
   - 国内 + 海外

3. **资产分散**
   - 股票 + 债券
   - 股票 + 商品
   - 股票 + 现金

### 3.3 对冲策略

**核心思想**：
- 用相关资产对冲风险
- 降低组合波动
- 在不确定性中寻找确定性

**常用方法**：

1. **期货对冲**
   - 股指期货对冲系统性风险
   - 商品期货对冲原材料风险

2. **期权对冲**
   - 买入认沽期权保护
   - 卖出认购期权增收

3. **配对交易**
   - 做多强势股
   - 做空弱势股
   - 获取相对收益

## 四、交易策略

### 4.1 选股策略

**价值选股**：
```
PE < 行业平均
PB < 行业平均
ROE > 15%
净利润持续增长
现金流优秀
```

**成长选股**：
```
营收增速 > 30%
利润增速 > 30%
市占率持续提升
行业空间大
```

**技术选股**：
```
均线多头排列
成交量放大
MACD金叉
RSI适中（40-70）
```

### 4.2 买入策略

**定投策略**：
- 定期定额投资
- 无视市场波动
- 长期坚持

**分批建仓**：
```
第一次：30%（建立底仓）
第二次：30%（回调加仓）
第三次：40%（确认后重仓）
```

**金字塔买入**：
```
价格低 → 买得多
价格高 → 买得少
成本摊低效果好
```

### 4.3 卖出策略

**止盈策略**：
- 达到预期收益（20-30%）考虑减仓
- 估值出现泡沫时分批卖出
- 高位放巨量可能是见顶信号

**卖出信号**：
```
1. 估值显著高估
2. 行业景气度下降
3. 公司竞争优势减弱
4. 发现更好的机会
5. 系统性风险来临
```
