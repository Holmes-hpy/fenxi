## 二、核心行业深度分析

### 2.1 半导体行业（801080.SI）

**行业特点**：
- 高技术壁垒、高研发投入
- 强周期性，但成长性更强
- 国产替代是长期逻辑

**产业链**：
```
设计 → 制造 → 封装 → 测试
  |      |       |
  v      v       v
华为海思  中芯国际  长电科技
高通     台积电    通富微电
英伟达   联发科    华天科技
```

**核心公司**：
- **设计**：华为海思、寒武纪、韦尔股份
- **制造**：中芯国际、华虹半导体
- **封装测试**：长电科技、通富微电、华天科技
- **设备**：北方华创、中微公司
- **材料**：沪硅产业、安集科技

**驱动因素**：
- 国产替代率提升
- 5G、AI、新能源车需求爆发
- 国家政策支持（半导体大基金）

**风险点**：
- 外部制裁风险
- 技术突破难度大
- 周期性波动

### 2.2 白酒行业（801090.SI）

**行业特点**：
- 强品牌壁垒
- 高毛利率、高现金流
- 抗通胀能力强

**竞争格局**：
```
高端：茅台、五粮液、国窖1573
次高端：洋河、汾酒、郎酒
中端：古井贡酒、今世缘
低端：牛栏山、老村长
```

**核心公司**：
- **贵州茅台**：酱香型、高端代表、提价能力强
- **五粮液**：浓香型、高端市场第二
- **泸州老窖**：浓香型鼻祖、国窖1573

**驱动因素**：
- 消费升级
- 品牌集中度提升
- 社交属性

**风险点**：
- 政策风险（禁酒令）
- 库存周期
- 消费观念转变

### 2.3 新能源汽车行业（801710.SI）

**行业特点**：
- 政策+市场双轮驱动
- 电动化+智能化+网联化
- 颠覆传统汽车产业链

**产业链**：
```
上游：锂、钴、镍等矿产资源
中游：电池、电机、电控
下游：整车制造
配套设施：充电桩、换电站
```

**核心公司**：
- **动力电池**：宁德时代、比亚迪、中创新航
- **整车**：比亚迪、特斯拉、蔚来、小鹏、理想
- **零部件**：拓普集团、三花智控
- **充电桩**：特锐德、许继电气

**驱动因素**：
- 碳中和目标
- 油价上涨
- 智能驾驶技术成熟

**风险点**：
- 补贴退坡
- 原材料价格波动
- 竞争加剧

### 2.4 银行行业（801710.SI）

**行业特点**：
- 强防御性
- 顺周期
- "经济周期之母"

**分类**：
| 类型 | 代表 | 特点 |
|-----|------|------|
| 国有大行 | 工、农、中、建、交 | 规模大、稳健 |
| 股份制银行 | 招行、兴业、平安 | 机制灵活、零售强 |
| 城商行 | 宁波、杭州、成都 | 区域性强、弹性大 |
| 农商行 | 江阴、无锡 | 规模小、区域性 |

**核心驱动**：
- 息差企稳
- 资产质量改善
- 财富管理转型

**风险点**：
- 房地产风险
- 地方融资平台风险
- 净息差收窄

## 三、行业分析框架

### 3.1 波特五力模型

| 五力 | 分析要点 |
|-----|---------|
| **行业内现有竞争者** | 竞争激烈程度、集中度 |
| **潜在进入者** | 进入壁垒高低 |
| **替代品** | 替代品威胁程度 |
| **供应商议价能力** | 供应商集中度、可替代性 |
| **购买者议价能力** | 客户集中度、信息透明度 |

### 3.2 行业生命周期

```
导入期 → 成长期 → 成熟期 → 衰退期
  |        |        |        |
高增长   快速增长   稳定增长   负增长
高风险   中风险    低风险    转型需求
```

**各阶段特征**：

**导入期**：
- 市场规模小
- 技术不成熟
- 风险高、机会大

**成长期**：
- 市场规模快速扩大
- 技术逐渐成熟
- 竞争加剧

**成熟期**：
- 市场规模稳定
- 竞争格局稳定
- 龙头效应明显

**衰退期**：
- 市场规模下降
- 替代品出现
- 转型或退出

### 3.3 行业配置时钟

| 经济阶段 | 行业配置 | 逻辑 |
|---------|---------|------|
| 复苏期 | 金融、周期、消费 | 经济回暖，业绩改善 |
| 过热期 | 周期、商品 | 通胀预期，商品涨价 |
| 滞胀期 | 现金防御 | 增长放缓，通胀高位 |
| 衰退期 | 债券、医药 | 降息预期，防御为主 |
