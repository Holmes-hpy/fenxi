## 二、基础构件速查

### 2.1 分型

| 分型 | 图形 | 含义 |
|-----|------|------|
| **顶分型** | 中间最高 | 上涨可能结束 |
| **底分型** | 中间最低 | 下跌可能结束 |

### 2.2 笔

| 笔 | 构成 | 说明 |
|---|------|------|
| **上升笔** | 底分型 + K线 + 顶分型 | 上涨 |
| **下降笔** | 顶分型 + K线 + 底分型 | 下跌 |

### 2.3 线段

- **定义**：连续三笔有重叠
- **特征**：比笔更稳定

### 2.4 中枢（核心）

```
        P2
       /  \
      /中枢\   P3
P1   /区间 \  /
 \  /_______\/
  \/       \/
    Q1 Q2

中枢区间 = [DD, GG]
DD：低点中的低点
GG：高点中的高点
```

## 三、走势类型

| 类型 | 特征 | 操作 |
|-----|------|------|
| **上涨** | 低点抬高，高点创新高 | 持有或买入 |
| **下跌** | 高点降低，低点创新低 | 观望或融券 |
| **盘整** | 高点相近，低点相近 | 高抛低吸 |

## 四、背驰判断

### 4.1 背驰类型

| 类型 | 条件 | 操作 |
|-----|------|------|
| **顶背驰** | 创新高但力度减弱 | 考虑卖出 |
| **底背驰** | 创新低但力度减弱 | 考虑买入 |

### 4.2 MACD辅助判断

```
顶背驰：
✓ 价格创新高
✗ MACD黄白线未创新高（或低于前高）
✗ 红柱面积缩小

底背驰：
✓ 价格创新低
✗ MACD黄白线未创新低（或高于前低）
✗ 绿柱面积缩小
```

## 七、缠论选股标准

### 7.1 买入条件

```
1. ✓ 周线在上涨或盘整中
2. ✓ 日线出现第一类或第二类买点
3. ✓ MACD在零轴上方
4. ✓ 均线多头排列
5. ✓ 有热点题材配合
```

### 7.2 排除条件

```
1. ✗ 周线在下跌中
2. ✗ 日线没有中枢支撑
3. ✗ 出现顶背驰信号
4. ✗ 连续无量涨停
```

## 八、操作纪律

### 8.1 核心原则

1. **只在大级别买点买入**
2. **严格止损**：跌破买点级别中枢下沿
3. **顺势而为**：上涨持有，下跌休息
4. **级别匹配**：小资金做30分钟，大资金做日线

### 8.2 止损设置

| 操作级别 | 止损建议 |
|---------|---------|
| 30分钟 | 5-8% |
| 日线 | 8-12% |
| 周线 | 15-20% |

## 十、常见问题

### 10.1 中枢如何画？

**步骤**：
1. 找到连续三笔
2. 确定三笔的重叠区间
3. 重叠区间即为中枢

### 10.2 背驰判断不准怎么办？

**解决方案**：
1. 结合大级别判断
2. 多指标验证（MACD + 均线）
3. 控制仓位，分批操作

### 10.3 三个买点冲突怎么办？

**优先级**：
1. 第二类买点最常用、最稳健
2. 第三类买点突破确认后操作
3. 第一类买点需要严格止损
