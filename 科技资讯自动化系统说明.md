# 🚀 科技前沿资讯自动化分析系统

## 📋 系统概述

这是一个每日上午9:30自动爬取科技前沿资讯的系统，专注于十五五规划相关产业（人工智能、半导体、新能源、数字经济等），分析资讯并提取知识，结合缠论进行知识沉淀，识别反向信号（自媒体标题党、资本雇佣枪手发文）。

## 🎯 核心功能

### 1. 📰 资讯爬取
- 每日定时爬取科技前沿资讯
- 关注十五五规划重点产业
- 多维度搜索和过滤

### 2. 🔍 智能分析
- **来源可靠性分析**：识别权威媒体和政府网站
- **核心信息提取**：自动提取事件主体、类型、影响范围
- **反向信号识别**：检测标题党、资本雇佣枪手特征
- **缠论视角分析**：结合技术分析识别买卖点

### 3. 🧠 知识沉淀
- 每日生成结构化分析报告
- 知识自动沉淀到知识库
- 历史数据可追溯

## 🚀 快速开始

### 方法一：立即执行一次（测试）

```bash
python3 auto_scheduler.py now
```

### 方法二：启动定时调度器（推荐）

```bash
python3 auto_scheduler.py start
```

调度器会在每个交易日（周一至周五）上午 9:30 自动执行任务。

### 方法三：直接运行主程序

```bash
python3 daily_tech_intel_system_v2.py
```

## ⏰ 定时任务配置

系统使用 cron 表达式配置：`30 9 * * 1-5`

- `30` - 第 30 分钟
- `9` - 上午 9 点
- `*` - 每天
- `*` - 每月
- `1-5` - 周一至周五

## 📊 报告结构

每日生成的报告包含以下章节：

### 1. 📌 报告摘要
- 爬取资讯总数
- 正常资讯数量
- 反向信号数量
- 高影响力资讯数量

### 2. 📋 十五五规划相关产业动态
按行业分类展示每条资讯，包含：
- 资讯标题和来源
- 核心内容摘要
- 事件类型和影响范围
- 缠论分析（买卖点信号）
- 风险预警

### 3. ⚠️ 反向信号预警
专门展示检测到的标题党和可疑资讯，包含：
- 风险等级评估
- 风险关键词列表
- 详细分析和建议

### 4. 📈 缠论视角深度分析
- 今日买卖点信号统计
- 核心要点回顾
- 策略建议（仓位管理、选股方向、入场时机）

### 5. 🧠 知识沉淀
- 今日知识点总结
- 后续关注方向

## 🔧 反向信号识别

系统通过以下特征识别可疑资讯：

| 类别 | 关键词示例 | 风险程度 |
|------|-----------|---------|
| 极端词汇 | 暴涨、暴跌、翻倍、妖股 | 高 |
| 内幕词汇 | 内幕、独家、重磅、震惊 | 高 |
| 主体词汇 | 主力、游资、机构、大佬 | 中 |
| 时间词汇 | 即将、马上、立即、刚刚 | 中 |
| 引导词汇 | 手把手、带你、躺赢、暴富 | 中 |
| 技术词汇 | 定律、战法、秘籍、神器 | 低 |
| 操作词汇 | 抄底、逃顶、上车、打板 | 中 |

## 🎓 缠论分析要点

### 第一类买点
- **特征**：政策发布、技术突破带来的底部机会
- **触发词**：政策底、技术底、估值底、重大利好、超跌反弹

### 第二类买点
- **特征**：第一类买点后的回调确认点
- **触发词**：回调企稳、二次确认、缩量回踩、支撑有效

### 第三类买点
- **特征**：突破中枢后的回抽确认
- **触发词**：趋势确立、放量突破、新高确认、强势上涨

## 📅 十五五规划重点关注领域

系统关注以下产业的最新动态：

1. **人工智能**：AI、大模型、深度学习、自动驾驶
2. **半导体**：芯片、集成电路、先进制程、国产替代
3. **新能源**：光伏、风电、储能、新能源汽车
4. **数字经济**：数据中心、算力、云计算、东数西算
5. **新材料**：稀土、碳纤维、石墨烯、第三代半导体
6. **生物医药**：创新药、基因编辑、mRNA

## 🛠️ 高级用法

### 修改定时任务时间

编辑 `auto_scheduler.py` 中的时间配置：

```python
scheduler.add_job(
    run_daily_tech_intel,
    trigger=CronTrigger(
        hour=9,      # 修改小时
        minute=30,   # 修改分钟
        day_of_week='mon-fri'  # 修改星期
    ),
    ...
)
```

### 自定义关注领域

编辑 `daily_tech_intel_system_v2.py` 中的 `FIFTEEN_FIVE_INDUSTRIES` 字典，添加或修改关注的领域和关键词。

### 添加更多来源

编辑 `RELIABLE_SOURCES` 列表，添加更多权威媒体和政府网站。

## 🔍 故障排查

### 问题：搜索工具不可用
**解决方案**：系统会自动使用示例数据，不影响其他功能运行。

### 问题：依赖缺失
**解决方案**：运行 `pip install apscheduler` 安装必要依赖。

### 问题：任务未执行
**解决方案**：检查电脑是否休眠，确保调度器程序正在运行。

## 💡 最佳实践

1. **工作日持续运行**：保持调度器在每个工作日运行
