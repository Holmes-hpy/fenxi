## 一、中国金融体系整体架构

### 1.1 五大核心主体

| 主体 | 作用 | 核心职能 |
|-----|------|---------|
| **央行** | 货币政策制定者 | 发行货币、调节利率、控制准备金 |
| **银行体系** | 货币派生主体 | 吸收存款、发放贷款、创造货币 |
| **财政体系** | 财政政策制定者 | 征税、支出、发债 |
| **金融市场** | 资源配置场所 | 股票、债券、外汇、衍生品市场 |
| **实体经济** | 最终服务对象 | 生产、消费、投资、进出口 |

### 1.2 资金流向图

```
          央行
           |
           v
     银行体系 ←→ 财政体系
       |    |
       v    v
  金融市场  实体经济
       |    |
       v    v
     房地产  制造业
     大宗    消费
     股票    投资
```

### 1.3 流动性三层体系

**第一层：基础货币（狭义流动性）**
- M0：流通中现金
- 来源：央行资产负债表
- 由央行直接控制

**第二层：银行体系流动性（狭义流动性）**
- 超额准备金率
- 反映银行间市场流动性
- 央行货币政策操作直接影响

**第三层：实体经济流动性（广义流动性）**
- 货币市场资金
- 信贷市场资金
- 资本市场资金
- 从银行间向实体传导

### 2.2 储备货币的构成

**储备货币 = 基础货币 = A - B**

- **A = 国外资产 + 对其他存款性公司债权 + 对其他金融性公司债权 + 其他资产**
- **B = 不计入储备货币的金融性公司存款 + 发行债券 + 国外负债 + 政府存款 + 自有资金 + 其他负债**

**Δ储备货币 = Δ国外资产 + Δ对其他存款性公司债权 + Δ对其他金融性公司债权 - Δ政府存款**

## 三、货币创造与供应量体系

### 3.1 基础货币与货币乘数

**货币创造过程**：
```
央行 → 基础货币（高能货币）
      ↓
银行 → 贷款 → 存款 → 新贷款 → 新存款 → ...
      ↓
派生货币 = 基础货币 × 货币乘数
```

**货币供应量层级**：
- **M0**：流通中现金（央行直接控制）
- **M1**：M0 + 单位活期存款（反映企业活力）
- **M2**：M1 + 定期存款 + 个人存款 + 其他存款（广义货币）

## 四、宏观经济学完整框架

### 4.1 总需求-总供给模型（AD-AS）

**总需求**：
- 消费（C）
- 投资（I）
- 政府购买（G）
- 净出口（NX）

**总供给**：
- 生产函数：Y = F(N, K)
- 短期：产量与就业
- 长期：潜在产出

**总需求函数**：
```
Y = c(i) + i(i) + g + nx(ε)
```

**总供给函数**：
- 古典：P=常数
- 常规：P随Y变化

### 4.2 IS-LM模型

**IS曲线（产品市场均衡）**：
```
IS: Y = c(Y-T) + i(r) + g
```

**LM曲线（货币市场均衡）**：
```
LM: M/P = L1(Y) + L2(r)
```

**财政政策效果**：
- IS陡峭 → 财政政策有效
- LM平坦 → 财政政策有效

**货币政策效果**：
- IS平坦 → 货币政策有效
- LM陡峭 → 货币政策有效

### 4.3 宏观经济学知识体系

```
总支出 → 消费/投资/政府购买/净出口
    ↓
总产出 → 短期波动/长期增长
    ↓
AS模型（生产函数）  IS模型（产品市场）
    ↓                ↓
劳动力市场         LM模型（货币市场）
    ↓                ↓
均衡点 (N, W/P)    均衡点 (Y, r)
    ↓                ↓
产出价格变动        利率变动
```

- 美国GDP下降46%

- 全球贸易萎缩60%

- 银行停业整顿
- 废除金本位
- 工业复兴法
- 农业调整法
- 建立社会保障制度

## 八、美国政策演变历史

### 8.1 冷战时期（1945-1991）

**布雷顿森林体系**：
- 美元与黄金挂钩
- 固定汇率制
- IMF、世界银行成立

**马歇尔计划**：
- 1948-1952年
- 美国援助欧洲130亿美元
- 重建欧洲经济

**尼克松冲击**（1971）：
- 停止美元兑换黄金
- 布雷顿森林体系崩溃
- 浮动汇率时代开始

### 8.2 80年代危机与转型

**沃尔克通胀治理**：
- 1979年保罗·沃尔克任美联储主席
- 大幅提高联邦基金利率至20%
- 成功控制通胀
- 但引发两次衰退

**里根经济学**：
- 大规模减税
- 放松管制
- 削减福利
- 增加国防支出

**1987年股灾**：
- 10月19日，道指下跌22.6%
- 美联储快速救市
- 避免系统性危机

### 8.3 90年代新经济

**信息革命**：
- 互联网兴起
- 科技股繁荣
- 经济持续增长

**1994年墨西哥危机**：
- 比索大幅贬值
- 美国牵头救助
- 500亿美元援助计划

**1997年亚洲金融危机**：
- 泰国货币危机开始
- 蔓延至东南亚、韩国
- IMF介入救助

### 8.4 21世纪危机与应对

**2000年互联网泡沫破裂**：
- 纳斯达克指数下跌78%
- 科技股大量破产
- 美联储降息救市

**2007-2009年次贷危机**：
- 房价下跌引发
- 次贷违约潮
- 雷曼兄弟破产
- 美联储量化宽松（QE）
- TARP救助计划
- 多德-弗兰克法案出台

**2010-2012年欧债危机**：
- 希腊、爱尔兰、葡萄牙、西班牙危机
- 欧洲央行OMT计划
- 欧洲稳定机制（ESM）

## 十、IS-LM模型与政策分析

### 10.1 财政政策效果

**IS曲线斜率**：
- 投资对利率敏感（d大）→ IS平坦 → 财政政策效果小
- 投资对利率不敏感（d小）→ IS陡峭 → 财政政策效果大

**LM曲线斜率**：
- 货币需求对利率敏感（h大）→ LM平坦 → 财政政策效果大
- 货币需求对利率不敏感（h小）→ LM陡峭 → 财政政策效果小

**挤出效应**：
- 财政支出增加 → 利率上升 → 私人投资减少
- 完全挤出（LM垂直）：财政政策完全无效
- 无挤出（LM水平）：财政政策完全有效

### 10.2 货币政策效果

**IS曲线斜率**：
- IS平坦 → 利率变动对投资影响大 → 货币政策效果好
- IS陡峭 → 利率变动对投资影响小 → 货币政策效果差

**LM曲线斜率**：
- LM陡峭 → 货币供给变动对利率影响大 → 货币政策效果好
- LM平坦 → 货币供给变动对利率影响小 → 货币政策效果差

### 10.3 案例：肯尼迪减税（1964-1966）

**背景**：

- 肯尼迪总统上任

**措施**：
- 降低公司税
- 降低个人所得税
- 投资税优惠

**结果**：
- GDP年均增速5.5%
- 失业率降至4.4%
- 被视为成功的财政政策案例
