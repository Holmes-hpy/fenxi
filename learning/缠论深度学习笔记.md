## 一、缠论概述

### 1.1 缠论的创立

**创始人**：缠中说禅（网络ID），真名李彪

**创立背景**：
- 2006-2008年在网络上连载
- 共108课，故称"缠论108课"
- 结合了中国传统文化与西方技术分析

**核心理念**：
> "走势终完美"——任何级别的任何走势类型，终要完成

### 1.2 缠论的特点

| 特点 | 说明 |
|-----|------|
| **几何化** | 用几何方法分析走势，不依赖指标 |
| **递归性** | 从最小单位逐级递归 |
| **完全分类** | 对走势进行完全分类 |
| **动力学结合** | 结合均线、MACD等判断背驰 |
| **实战性** | 强调可操作性，有明确的买卖点 |

### 1.3 缠论的理论框架

```
分型 → 笔 → 线段 → 中枢 → 走势类型 → 买卖点
  ↓      ↓      ↓       ↓        ↓
最小   基础   中级   核心    三类买卖点
单位   构建   构建   概念
```

## 二、缠论基础概念

### 2.1 分型（最基础元素）

#### 顶分型
```
    K线3（最高）
   / \
K线1  K线2
```
- 三根K线组成
- 中间一根K线最高
- 第二根K线在左右肩之上
- 预示上涨可能结束

#### 底分型
```
K线1  K线2
   \ /
    K线3（最低）
```
- 三根K线组成
- 中间一根K线最低
- 第二根K线在左右肩之下
- 预示下跌可能结束

#### 分型的构成条件
```
顶分型：第1、3根K线不包含第2根K线
底分型：第1、3根K线不包含第2根K线

包含关系处理：
- 上升趋势：取高高
- 下降趋势：取低低
```

### 2.2 笔（连接的构件）

#### 笔的定义
- **笔**：连接两个相邻的分型
- **上升笔**：底分型 + 至少一根独立K线 + 顶分型
- **下降笔**：顶分型 + 至少一根独立K线 + 底分型

#### 笔的规则
```
笔的构成条件：
1. 顶分型 + K线 + 底分型（上升笔）
2. 底分型 + K线 + 顶分型（下降笔）
3. 连续顶分型之间不构成笔
4. 连续底分型之间不构成笔
5. 笔的起始分型被破坏时，笔不成立
```

#### 笔的划分步骤
1. 找到所有分型
2. 处理包含关系
3. 连接符合条件的分型
4. 形成笔

### 2.3 线段（更稳定的结构）

#### 线段的定义
- **线段**：连续的三笔，且三笔有重叠
- 线段比笔更稳定
- 是构建中枢的基础

#### 线段的特征
```
线段必须满足：
1. 由奇数笔构成（3、5、7...）
2. 连续三笔必须有重叠
3. 线段被线段破坏（反向线段走出3笔）

线段破坏：
- 上升线段被下降线段破坏
- 下降线段被上升线段破坏
```

### 2.4 走势的递归

```
最小级别走势：
1分钟K线 → 1分钟笔 → 1分钟线段 → 1分钟中枢 → 1分钟走势类型

递归级别：
1分钟 → 5分钟 → 30分钟 → 日线 → 周线 → 月线

每个级别都遵循相同的递归规则
```

## 三、缠论核心：中枢

### 3.1 中枢的定义

**中枢**：走势中至少连续三笔（段）发生重叠的区域

```
        P2
       /  \
      /    \
P1   /中枢\   P3
 \  /区间 \  /
  \/_______\/
    Q1 Q2
```

**中枢区间**：
- 上界：P1、P3的高点中的较高者
- 下界：Q1、Q3的低点中的较低者
- 中枢区间：[Q1-Q2, P1-P2]

### 3.2 中枢的构成

| 构成要素 | 说明 |
|---------|------|
| **进入段** | 进入中枢的第一笔（段） |
| **中枢本体** | 三笔（段）重叠的部分 |
| **离开段** | 离开中枢的最后一笔（段） |

#### 中枢的画法
```
1. 找到连续三笔（段）
2. 确定三笔（段）的重叠区间
3. 重叠区间即为中枢区间
4. 中枢区间 = [GG, DD]
   - GG：高点中的高点
   - DD：低点中的低点
```

### 3.3 中枢的级别

| 级别 | 说明 | 震荡时间 |
|-----|------|---------|
| **1分钟中枢** | 1分钟图上的中枢 | 15-30分钟以上 |
| **5分钟中枢** | 5分钟图上的中枢 | 1小时以上 |
| **30分钟中枢** | 30分钟图上的中枢 | 2-4小时以上 |
| **日线中枢** | 日线图上的中枢 | 数天到数周 |
| **周线中枢** | 周线图上的中枢 | 数周到数月 |

### 3.4 中枢的性质

| 性质 | 说明 |
|-----|------|
| **引力作用** | 价格向中枢靠拢 |
| **震荡区间** | 中枢内价格震荡 |
| **方向指示** | 突破方向指示趋势 |
| **支撑压力** | 中枢边界形成支撑压力 |

### 3.5 中枢的延伸与扩展

**中枢延伸**：
- 中枢完成后，继续走出9笔以上延伸
- 中枢不断延伸，震荡加剧

**中枢扩展**：
- 围绕原中枢震荡，形成更大级别的中枢
- 原中枢升级

**中枢破坏**：
- 离开段力度不足，返回中枢
- 离开段力度强，突破中枢

## 四、走势类型

### 4.1 三种走势类型

| 走势类型 | 定义 | 特征 |
|---------|------|------|
| **上涨走势** | 依次上升的走势 | 低点不断抬高，高点不断创新高 |
| **下跌走势** | 依次下降的走势 | 高点不断降低，低点不断创新低 |
| **盘整走势** | 在一个区间内震荡 | 高点相近，低点相近 |

### 4.2 上涨走势

```
定义：
- 走势由一个上涨中枢 + 离开段组成
- 或者由多个同级别上涨中枢组成

特征：
- 低点不断抬高
- 高点不断创新高
- 每次回调都在前次回调之上

示意图：
    /\
   /  \    /\
  /    \  /  \
 /      \/    \
/              \
```

### 4.3 下跌走势

```
定义：
- 走势由一个下跌中枢 + 离开段组成
- 或者由多个同级别下跌中枢组成

特征：
- 高点不断降低
- 低点不断创新低
- 每次反弹都在前次反弹之下

示意图：
\              /
 \    /\      /
  \  /  \    /
   \/    \  /
           \/
```

### 4.4 盘整走势

```
定义：
- 走势围绕一个中枢震荡
- 高点相近，低点相近

特征：
- 无明显方向
- 在固定区间内波动
- 震荡完毕后选择方向

示意图：
   _____上限_____
  /               \
 /                 \
/                   \
\                   /
  \               /
    -----下限-----
```

### 4.5 走势的多义性

**走势多义性**：
- 同一段走势可以有多种划分方法
- 需要结合中枢、级别判断
- 没有唯一正确答案

**处理方法**：
1. 结合走势中枢判断
2. 结合均线系统判断
3. 结合动力学判断

## 五、背驰理论

### 5.1 背驰的定义

**背驰**：走势在继续创新高（新低）时，力度明显减弱

| 类型 | 说明 |
|-----|------|
| **顶背驰** | 上涨创新高，但力度减弱 |
| **底背驰** | 下跌创新低，但力度减弱 |

### 5.2 背驰的判断方法

#### 方法一：趋势力度比较

**趋势力度** = 进攻力度 - 撤退力度

```
上涨趋势力度：
力度1 = 高点1 - 低点1
力度2 = 高点2 - 低点2
力度3 = 高点3 - 低点3

背驰判断：
力度3 < 力度2 < 力度1 → 可能背驰
```

#### 方法二：MACD辅助判断

```
顶背驰判断：
1. 价格创新高（高于前高点）
2. MACD黄白线未创新高（或低于前高）
3. 红柱子面积缩小

底背驰判断：
1. 价格创新低（低于前低点）
2. MACD黄白线未创新低（或高于前低）
3. 绿柱子面积缩小
```

#### 方法三：均线辅助判断

```
判断方法：
- 用5日均线和10日均线
- 吻（交叉）的力度判断

飞吻：5日均线和10日均线只是稍微接触
唇吻：5日均线和10日均线接触力度较大
湿吻：5日均线和10日均线交叉缠绕
```

### 5.3 背驰的级别

| 级别 | 说明 | 准确性 |
|-----|------|--------|
| **小级别背驰** | 1分钟、5分钟背驰 | 准确率较低 |
| **中级别背驰** | 30分钟、日线背驰 | 准确率中等 |
| **大级别背驰** | 周线、月线背驰 | 准确率较高 |

### 5.4 背驰与买卖点

**背驰的实战意义**：
- 顶背驰 → 可能转折 → 考虑卖出
- 底背驰 → 可能反弹 → 考虑买入
- 背驰级别越大，准确性越高

**背驰陷阱**：
- 小级别背驰可能失败
- 需要结合大级别判断
- 背驰不是绝对的

## 六、三类买卖点

### 6.1 第一类买卖点（精确抄底逃顶）

**第一类买点**：某级别下跌走势背驰后，形成的买点

```
位置：前一下跌走势的结束点
特征：
- 下跌走势的最后一个中枢下方
- 背驰段创新低后回升
- MACD底背驰
```

**第一类卖点**：某级别上涨走势背驰后，形成的卖点

```
位置：前一下跌走势的结束点
特征：
- 上涨走势的最后一个中枢上方
- 背驰段创新高后回落
- MACD顶背驰
```

#### 第一类买卖点示意图
```
买点：
          ____中枢____
         /            \
        /              \
-------/                \----
      ↑                  ↑
    背驰段            第一买点

卖点：
-----\                /-----
       \              /
        \            /
         ____中枢____
              ↑                  ↑
          第一卖点            背驰段
```

### 6.2 第二类买卖点（稳健买入）

**第二类买点**：第一类买点后，第一次回落形成的买点

```
位置：第一类买点之后，股价回落但不创新低
特征：
- 低于第一类买点
- 但不创新低
- 可能在中枢下方或内部
```

**第二类卖点**：第一类卖点后，第一次反弹形成的卖点

```
位置：第一类卖点之后，股价反弹但不创新高
特征：
- 高于第一类卖点
- 但不创新高
- 可能在中枢上方或内部
```

### 6.3 第三类买卖点（突破确认）

**第三类买点**：突破中枢后，回调不进入中枢形成的买点

```
位置：中枢上方
特征：
- 突破中枢后回调
- 回调低点不进入中枢区间
- 确认突破有效
```

**第三类卖点**：跌破中枢后，反弹不进入中枢形成的卖点

```
位置：中枢下方
特征：
- 跌破中枢后反弹
- 反弹高点不进入中枢区间
- 确认跌破有效
```

### 6.4 三类买卖点对比

| 类型 | 位置 | 风险 | 收益 | 准确率 |
|-----|------|------|------|--------|
| **第一类** | 转折点 | 高 | 大 | 中 |
| **第二类** | 回调低点 | 中 | 中 | 高 |
| **第三类** | 突破后 | 低 | 中 | 高 |

### 6.5 买卖点的级别选择

| 资金量 | 建议级别 | 说明 |
|-------|---------|------|
| **小资金** | 5分钟、30分钟 | 灵活，但需盯盘 |
| **中等资金** | 30分钟、日线 | 兼顾灵活性和稳定性 |
| **大资金** | 日线、周线 | 稳定，但收益较慢 |

## 七、缠论动力学

### 7.1 均线系统

#### 均线吻系统

| 吻的类型 | 说明 | 实战意义 |
|---------|------|---------|
| **飞吻** | 5日均线和10日均线只是稍微接触 | 短期调整 |
| **唇吻** | 5日均线和10日均线接触力度较大 | 中期调整 |
| **湿吻** | 5日均线和10日均线交叉缠绕 | 可能产生大行情 |

#### 均线操作法

```
湿吻后趋势判断：
1. 湿吻向上发散 → 上涨趋势
2. 湿吻向下发散 → 下跌趋势
3. 湿吻缠绕横走 → 盘整趋势
```

### 7.2 MACD辅助

#### MACD参数设置

| 参数 | 默认值 | 缠论推荐 |
|-----|--------|---------|
| 快线EMA | 12 | 12 |
| 慢线EMA | 26 | 26 |
| 信号线 | 9 | 9 |

#### MACD判断方法

**零轴判断**：
- 在零轴上方 → 多头市场
- 在零轴下方 → 空头市场
- 在零轴附近 → 震荡市场

**柱子判断**：
- 红柱伸长 → 上涨力度增强
- 红柱缩短 → 上涨力度减弱
- 绿柱伸长 → 下跌力度增强
- 绿柱缩短 → 下跌力度减弱

### 7.3 力度判断

#### 趋势力度

**趋势力度定义**：
```
趋势力度 = 进攻力度 - 撤退力度

上涨趋势力度：
= (本次上涨幅度) - (上次下跌幅度)

下跌趋势力度：
= (本次下跌幅度) - (上次上涨幅度)
```

#### 力度衰减

**判断方法**：
```
1. 比较相邻两段趋势的力度
2. 力度逐渐减弱 → 背驰可能形成
3. 力度突然减弱 → 背驰确认
```

### 7.4 量价关系

#### 量价配合

| 情况 | 说明 | 信号 |
|-----|------|------|
| **价升量增** | 上涨时成交量放大 | 正常，后市看好 |
| **价升量缩** | 上涨时成交量萎缩 | 背驰信号 |
| **价跌量增** | 下跌时成交量放大 | 可能加速下跌 |
| **价跌量缩** | 下跌时成交量萎缩 | 底部信号 |

## 八、实战应用

### 8.1 选股标准

| 标准 | 说明 |
|-----|------|
| **走势结构** | 处于上涨或盘整中，不在下跌中 |
| **中枢位置** | 在大级别中枢上方，强势 |
| **题材概念** | 有热点题材配合 |
| **基本面** | 业绩改善或稳定 |

### 8.2 操作策略

#### 日线级别操作

```
选股：
1. 周线在上涨或盘整中
2. 日线出现第一类或第二类买点
3. 有热点题材配合

买入：
1. 第一类买点（激进）：底背驰后买入
2. 第二类买点（稳健）：回调不创新低买入
3. 第三类买点（突破）：突破确认后买入

止损：
1. 跌破买入级别中枢下沿
2. 止损幅度：5-8%

持有：
1. 在上涨走势中持有
2. 每次回调不破前低，持有
3. 出现背驰，考虑减仓

卖出：
1. 上涨走势背驰后卖出
2. 跌破日线上升趋势线
3. 达到目标位
```

### 8.3 风险控制

| 原则 | 说明 |
|-----|------|
| **不追高** | 只在买点买入，不追涨 |
| **止损明确** | 买入前设定止损位 |
| **仓位管理** | 分批建仓，不满仓 |
| **级别匹配** | 操作级别与资金量匹配 |
| **顺势而为** | 上涨持有，下跌休息 |

### 8.4 常见错误

| 错误 | 说明 | 改正 |
|-----|------|------|
| **追涨杀跌** | 在卖点买、买点卖 | 只在买点买，卖点卖 |
| **不止损** | 亏损死扛 | 严格止损 |
| **频繁操作** | 级别混乱 | 固定操作级别 |
| **重仓** | 满仓一只 | 分仓配置 |
| **逆势** | 下跌中抄底 | 顺势而为 |

## 九、缠论与现代技术分析对比

### 9.1 缠论 vs 波浪理论

| 对比项 | 波浪理论 | 缠论 |
|-------|---------|------|
| **起源** | 国外 | 中国 |
| **基础** | 5浪上升3浪下降 | 中枢+走势类型 |
| **数浪** | 有主观性 | 有客观标准 |
| **买点** | 不明确 | 三类买点明确 |
