# 缠论实战检查清单

**创建时间**：2026-05-25  
**用途**：实战操作时对照检查

---

## 一、买入前检查清单

### 1.1 大级别方向检查

| 检查项 | 是/否 | 说明 |
|-------|------|-----|
| 周线趋势是否向上 | ☐ | 大级别定方向 |
| 日线趋势是否向上 | ☐ | 次级别确认 |
| 是否在大级别中枢上方 | ☐ | 强势 |
| 是否在大级别中枢下方 | ☐ | 弱势 |

### 1.2 买点类型检查

| 检查项 | 是/否 | 说明 |
|-------|------|-----|
| 是否是第二类买点（优先） | ☐ | 最稳健，推荐 |
| 是否是第一类买点 | ☐ | 精确但风险高 |
| 是否是第三类买点 | ☐ | 突破确认 |

### 1.3 技术指标检查

| 检查项 | 是/否 | 说明 |
|-------|------|-----|
| MACD是否在零轴上方 | ☐ | 强势 |
| MACD是否金叉 | ☐ | 买入信号 |
| 均线是否多头排列 | ☐ | 趋势向上 |
| 成交量是否配合 | ☐ | 放量上涨更可靠 |
| RSI是否在合理区间 | ☐ | 不超买超卖 |

### 1.4 缠论结构检查

| 检查项 | 是/否 | 说明 |
|-------|------|-----|
| 是否有明确的中枢 | ☐ | 中枢是核心 |
| 走势结构是否健康 | ☐ | 看走势类型 |
| 背驰是否确认（如有） | ☐ | MACD辅助 |

### 1.5 风险控制检查

| 检查项 | 是/否 | 说明 |
|-------|------|-----|
| 是否设置了止损位 | ☐ | 必须！ |
| 止损位是否合理 | ☐ | 不要太近太远 |
| 仓位是否合理 | ☐ | 单只不超过30% |
| 总仓位是否合适 | ☐ | 根据市场环境 |

## 二、持有中检查清单

### 2.1 走势健康检查

| 检查项 | 是/否 | 说明 |
|-------|------|-----|
| 趋势是否保持向上 | ☐ | 看均线 |
| 中枢是否稳定 | ☐ | 围绕中枢震荡是正常的 |
| 是否有背驰信号 | ☐ | 预警信号 |
| 成交量是否正常 | ☐ | 异常放量要注意 |

### 2.2 大级别变化检查

| 检查项 | 是/否 | 说明 |
|-------|------|-----|
| 周线趋势是否改变 | ☐ | 重要！ |
| 日线趋势是否改变 | ☐ | 重要！ |
| 是否有重大利空 | ☐ | 政策、消息等 |

### 2.3 止损调整检查

| 检查项 | 是/否 | 说明 |
|-------|------|-----|
| 是否需要提高止损位 | ☐ | 盈利后可以保护利润 |
| 止损位是否被触及 | ☐ | 触及必须执行！ |

## 三、卖出检查清单

### 3.1 卖点出现检查

| 检查项 | 是/否 | 说明 |
|-------|------|-----|
| 是否出现第一类卖点 | ☐ | 背驰后 |
| 是否出现第二类卖点 | ☐ | 最稳健 |
| 是否出现第三类卖点 | ☐ | 跌破确认 |

### 3.2 技术指标检查

| 检查项 | 是/否 | 说明 |
|-------|------|-----|
| MACD是否死叉 | ☐ | 卖出信号 |
| MACD是否顶背驰 | ☐ | 重要信号 |
| 均线是否空头排列 | ☐ | 趋势向下 |
| RSI是否超买 | ☐ | 预警信号 |

### 3.3 止损执行检查

| 检查项 | 是/否 | 说明 |
|-------|------|-----|
| 是否触及止损位 | ☐ | 触及必须卖！ |
| 是否有重大利空 | ☐ | 考虑减仓或清仓 |
| 是否达到目标位 | ☐ | 考虑减仓或清仓 |

## 四、注意事项

### 4.1 重要原则

1. **不要预测，只看信号！**
2. **买点买，卖点卖！**
3. **严格执行止损！**
4. **不要有侥幸心理！**
5. **耐心等待机会！**

### 4.2 常见错误避免

❌ 不要追高！
❌ 不要看到背驰就激动！
❌ 不要不止损！
❌ 不要频繁操作！
❌ 不要满仓！

---

**使用方法**：每次操作前，对照检查清单逐项确认！
