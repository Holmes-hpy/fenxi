## 三、分型：图形识别

### 3.1 顶分型图形详解

#### 顶分型的图形

```
【标准顶分型】
    K线3（最高）
   /   \
K线1   K线2
```

**图形特征**（必须同时满足！）：
1. **三根K线组成**
2. **中间K线最高**：K线3的高点 > K线1、K线2的高点
3. **不包含关系**：K线1和K线2都不能完全包含K线3

**顶分型的含义**：
- 上涨可能结束的信号
- 但不一定形成顶部，需要结合后续走势判断

#### 顶分型的变形

**顶分型1（最标准）**：
```
    /\
   /  \
  /    \
```

**顶分型2（略变形）**：
```
      /\
     /  \
    /    \
```

**顶分型3（右侧略长）**：
```
    /\
   /  \
  /    \
       \
```

### 3.2 底分型图形详解

#### 底分型的图形

```
【标准底分型】
K线1   K线2
   \  /
    K线3（最低）
```

**图形特征**（必须同时满足！）：
1. **三根K线组成**
2. **中间K线最低**：K线3的低点 < K线1、K线2的低点
3. **不包含关系**：K线1和K线2都不能完全包含K线3

**底分型的含义**：
- 下跌可能结束的信号
- 但不一定形成底部，需要结合后续走势判断

#### 底分型的变形

**底分型1（最标准）**：
```
    \  /
     \/
```

**底分型2（略变形）**：
```
    \    /
     \  /
      \/
```

**底分型3（右侧略长）**：
```
    \
     \  /
      \/
```

### 3.3 包含关系处理（关键！容易出错！）

#### 什么是包含关系？

**包含关系定义**：
一根K线的高低点完全在另一根K线的高低点范围内

```
【包含关系示例】
K线A：  K线B：
  /\       /\
 /  \     /  \
/    \   /    \
       \/
```

**K线B完全包含K线A**：
- K线B高点 > K线A高点
- K线B低点 < K线A低点

#### 包含关系处理原则

**原则1：先确定方向**
- 上升趋势：看前一根K线是上升还是下降
- 下降趋势：看前一根K线是上升还是下降

**原则2：取法规则**
- **上升趋势**：取高高（高点更高，低点更高）
- **下降趋势**：取低低（高点更低，低点更低）

#### 包含关系处理示例

```
【示例1：上升趋势中的包含】
K线1： /\
K线2： /\ （包含K线1）

处理：取高高！
  高点=K线2高点
  低点=K线2低点

【示例2：下降趋势中的包含】
K线1： \/
K线2： \/ （包含K线1）

处理：取低低！
  高点=K线2低点
  低点=K线2低点
```

**关键要点**：
- 包含关系是缠论中最容易出错的地方！
- 必须先判断趋势方向，再决定取法
- 处理包含关系是画笔画段的前提！

## 四、笔：图形识别

### 4.1 笔的定义

**笔的构成**：
- 底分型 + 1+独立K线 + 顶分型 → 上升笔
- 顶分型 + 1+独立K线 + 底分型 → 下降笔

**关键要点**：
- **至少有一根独立K线**：不是必须一根，一根或多根都可以
- **顶底分型要分明**：顶必须比底高，底必须比顶低

### 4.2 上升笔图形详解

```
【标准上升笔】
    K线1（底分型）
     /
    K线2（独立K线）
       \
      K线3（顶分型）
```

**上升笔的确认条件**：
1. 底分型和顶分型都标准
2. 至少有一根独立K线连接
3. 顶分型最高点 > 底分型最低点

### 4.3 下降笔图形详解

```
【标准下降笔】
    K线1（顶分型）
      \
      K线2（独立K线）
     /
    K线3（底分型）
```

**下降笔的确认条件**：
1. 顶分型和底分型都标准
2. 至少有一根独立K线连接
3. 顶分型最高点 > 底分型最低点

### 4.4 笔的识别误区

**误区1：顶底连在一起，没有独立K线**
```
错误示例：
    /\
   /  \
  /    \
   \  /
    \/

问题：顶分型和底分型直接连接，没有独立K线，不构成笔！
```

**误区2：包含关系没有处理好**
```
错误示例：
    /\
   /  \
  /    \
       \
  /\    /
 /  \  /

问题：包含关系没有处理，导致画笔错误！
```

## 五、线段：图形识别

### 5.1 线段的定义

**线段构成**：
- 至少连续的三笔
- 这三笔必须有重叠区间
- 线段由奇数笔构成（3笔、5笔、7笔...）

**线段比笔更稳定**：
- 笔是短期波动，线段是中期走势
- 线段是构成中枢的基础

### 5.2 标准线段图形

```
【上升线段】
  笔1    笔2    笔3
 /\  /\  /\
/  \/  \/  \

特征：高点不断抬高，低点不断抬高
```

```
【下降线段】
\  /\  /\
 \/  \/  \
  笔1    笔2    笔3

特征：高点不断降低，低点不断降低
```

```
【盘整线段】
    /\  /\  /\
   /  \/  \/  \
  /              \
 /                \

特征：在一个区间内震荡
```

### 5.3 线段被破坏的图形

```
【上升线段被下降线段破坏】
     /\  /\
    /  \/  \
   /    /\  \
  /    /  \  \
 上升   下降

关键：反向线段走出3笔，且有重叠！
```

**线段被破坏的意义**：
- 确认原线段结束
- 新线段开始
- 这是判断转折的重要依据！

## 六、缠论核心：中枢（图形详解）

### 6.1 中枢定义（最核心！）

**中枢定义**：走势中至少连续三笔发生重叠的区域

```
【标准中枢图形】
    高点2
   /\
  /  \ 中枢区间
 /    \---
/      \--
\      /--
 \    /---
  \  /----
   \/-----
    低点1  低点2
```

**中枢区间的确定**：
- **上沿**：三个高点中的最高值
- **下沿**：三个低点中的最低值
- **中枢区间**：[下沿, 上沿]

### 6.2 中枢的图形详解

#### 上升中枢图形

```
【上升中枢】
   /\  /\
  /  \/  \
 /    /\  \
/    /  \  \

特征：
- 高点逐步抬高
- 低点逐步抬高
- 重叠区域形成中枢
```

#### 下降中枢图形

```
【下降中枢】
\      /
 \    /
  \  /
   \/

特征：
- 高点逐步降低
- 低点逐步降低
- 重叠区域形成中枢
```

### 6.3 中枢的延伸与扩展

**中枢延伸**：
- 中枢形成后继续震荡
- 9笔及以上的延伸需要注意级别升级

**中枢扩展**：
- 围绕原中枢震荡
- 形成更大级别的中枢
- 级别升级！

## 八、背驰理论：图形识别

### 8.1 背驰定义

**背驰**：走势继续创新高（新低），但力度明显不足

**图形识别要点**：
- 看两段走势的力度对比
- 结合MACD辅助判断
- 不要只看价格，要看力度！

### 8.2 顶背驰图形详解

```
【顶背驰图形】
        高点B
       /\
      /  \
     /    \
    /高点A\
   /        \
  /          \
 /            \
```

**顶背驰确认条件**（必须同时满足！）：
1. **价格新高**：高点B > 高点A
2. **力度不足**：
   - 走势B的角度比走势A平缓
   - 走势B的长度比走势A短
3. **MACD辅助**：
   - MACD黄白线不创新高
   - MACD红柱面积缩小

**顶背驰实战意义**：
- 上升走势可能结束
- 考虑减仓或卖出
- 但不是绝对的，需要结合级别！

### 8.3 底背驰图形详解

```
【底背驰图形】
\            /
 \          /
  \低点B  /
   \      /
    \低点A/
     \    /
      \  /
       \/
```

**底背驰确认条件**（必须同时满足！）：
1. **价格新低**：低点B < 低点A
2. **力度不足**：
   - 走势B的角度比走势A平缓
   - 走势B的长度比走势A短
3. **MACD辅助**：
   - MACD黄白线不创新低
   - MACD绿柱面积缩小

**底背驰实战意义**：
- 下跌走势可能结束
- 考虑买入或建仓
- 但不是绝对的，需要结合级别！

### 8.4 MACD辅助背驰判断图形

```
【MACD顶背驰】
价格走势：
    新高
   /\
  /  \
 /    \

MACD柱：
    [][] 面积A
     []

     []   面积B
    []

关键：面积B < 面积A！
```

**MACD背驰判断步骤**：
1. 看价格是否创新高/新低
2. 看MACD黄白线是否创新高/新低
3. 看MACD柱子面积是否缩小
4. 综合判断！

## 九、三类买卖点：图形识别

### 9.1 第一类买卖点（精确抄底逃顶）

#### 第一类买点图形

```
【第一类买点】
\      /
 \    /
  \  /
   \/ 背驰点

位置：前下跌走势结束的背驰点
特点：抄底，精确但风险高
```

**第一类买点确认条件**：
1. 前走势是下降走势
2. 出现底背驰
3. 背驰后转折

**操作要点**：
- 激进的可以在背驰点买入
- 稳健的等待第二类买点
- 必须设止损！

#### 第一类卖点图形

```
【第一类卖点】
        /\
       /  \
      /    \
     /      \
    /        \
     /背驰点
```

**第一类卖点确认条件**：
1. 前走势是上升走势
2. 出现顶背驰
3. 背驰后转折

**操作要点**：
- 激进的可以在背驰点卖出
- 稳健的等待第二类卖点
- 不要贪心！

### 9.2 第二类买卖点（最稳健！）

#### 第二类买点图形（重点！）

```
【第二类买点图形（最常用！）】
\      /
 \    /
  \  /
   \/ 第一买点
   /\ 第二类买点！← 最稳健！
  /  \
```

**第二类买点确认条件**（必须同时满足！）：
1. 出现第一类买点
2. 股价回调
3. **回调不创新低！**（关键！）
4. 可能在中枢下方或内部

**第二类买点的优势**：
- **最稳健！**（比第一类安全）
- **最常用！**（适合大多数人）
- **成功率高！**（历史数据验证）

**操作要点**：
- 耐心等待回调
- 确认不创新低
- 果断买入

#### 第二类卖点图形

```
【第二类卖点图形】
        /\
       /  \
      /    \
     /      \
    /        \
   /第一卖点
  /
   \ 第二类卖点
    \
```

**第二类卖点确认条件**：
1. 出现第一类卖点
2. 股价反弹
3. **反弹不创新高！**（关键！）

### 9.3 第三类买卖点（突破确认）

#### 第三类买点图形

```
【第三类买点图形】
     /\
    /  \
   /    \
  /      \ 第三买点
 /中枢  \ （回踩不进中枢！）
\      /
 \    /
  \  /
   \/
```

**第三类买点确认条件**：
1. 突破前中枢
2. 回踩回调
3. **回踩不进入中枢！**（关键！）
4. 确认突破有效

**第三类买点实战意义**：
- 确认突破后买入
- 趋势确立后参与
- 避免假突破！

#### 第三类卖点图形

```
【第三类卖点图形】
\      /
 \    /
  \  /
   \/
   /\
  /  \
 /    \
/      \ 第三卖点（反弹不进中枢！）
\      /
```

**第三类卖点确认条件**：
1. 跌破前中枢
2. 反弹回抽
3. **反弹不进入中枢！**（关键！）
4. 确认跌破有效

### 9.4 三类买卖点对比总结

| 买卖点 | 位置 | 风险 | 收益 | 特点 | 推荐度 |
|-------|------|------|------|------|--------|
| **第一类** | 背驰点 | 高 | 大 | 精确但风险高 | ⭐⭐⭐ |
| **第二类** | 回调点 | 中 | 中 | **最稳健最常用！** | ⭐⭐⭐⭐⭐ |
