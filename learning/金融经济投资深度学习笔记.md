## 一、经济学、金融学、投资学的关系

### 1.1 三者的本质定义

| 学科 | 本质 | 核心问题 |
|-----|------|---------|
| **经济学** | 研究稀缺资源如何配置 | 生产什么？如何生产？为谁生产？ |
| **金融学** | 研究资金的募集与投资 | 如何融资？如何定价？如何管理风险？ |
| **投资学** | 在不确定条件下进行资产配置 | 选什么资产？权重多少？如何控制风险？ |

### 1.2 三者的层次关系

```
经济学（宏观环境）
    ↓ 提供基础框架和分析工具
金融学（资金融通）
    ↓ 提供定价模型和投资工具
投资学（资产配置）
    ↓ 引导资金流向
实体经济（资源配置）
    ↓ 反过来影响经济周期
宏观经济（新一轮循环）
```

### 1.3 第一性原理的视角

**传统思维**：
- 经济学：研究GDP、CPI、失业率
- 金融学：研究PE、PB、ROE
- 投资学：研究K线、MACD、RSI

**第一性原理思维**：
- 经济学的本质：稀缺资源的有效配置
- 金融学的本质：资金的时间价值和风险定价
- 投资学的本质：用今天的现金换取未来更多的现金

## 二、金融学核心课程体系

### 2.1 十大核心课程详解

#### 1. 微观经济学：金融决策的"底层逻辑锚"

**核心价值**：
- 金融的本质是"资源定价与配置"
- 微观经济学讲透"价格怎么形成、决策怎么做出"
- 股票涨跌、期权定价、企业融资成本，本质都是"供需均衡"、"边际效用"等概念的具象化

**关键概念**：
- 供需均衡
- 边际效用
- 成本曲线
- 市场竞争格局

**实战应用**：
- 分析某品牌涨价的原因（供给端 or 需求端）
- 分析企业扩产的决策（成本曲线与市场格局）

#### 2. 宏观经济学：判断趋势的"市场晴雨表"

**核心价值**：
- 金融永远跟着经济周期走
- GDP、CPI、货币政策工具是预判市场方向的"信号"
- 不懂宏观，做投资就是"盲人摸象"

**关键指标**：
- GDP增速
- CPI通胀率
- 货币政策工具（降准、降息、公开市场操作）
- 财政政策（赤字率、专项债）

**实战应用**：
- 央行降准 → 宽松货币政策 → 流动性增加 → 利好股市
- 政府发专项债 → 财政扩张 → 基建受益 → 相关板块机会

#### 3. 货币银行学：金融体系的"解剖说明书"

**核心价值**：
- 讲清"钱在金融体系里的流转逻辑"
- 商业银行如何吸储放贷
- 央行怎么用准备金率调控市场

**关键机制**：
- 存款准备金制度
- 货币乘数效应
- 央行货币政策传导机制

**实战应用**：
- 降准0.5% → 释放流动性 → M2增加
- 理解银行股的核心逻辑（息差、坏账率）

#### 4. 国际金融学：跨境金融的"实操手册"

**核心价值**：
- 外资投行做跨境并购
- 券商做港股美股业务
- 企业对冲汇率风险

**关键知识**：
- 外汇远期
- 国际收支平衡
- 汇率决定理论（购买力平价、利率平价）

#### 5. 金融市场学：各类资产的"玩法指南"

**核心价值**：
- 股票、债券、基金、期货的"规则差异"
- A股和港股交易时间差异
- 债券久期怎么影响收益

**关键概念**：
- 交易机制
- 定价机制
- 市场结构

#### 6. 公司金融：企业金融的"核心技能包"

**核心价值**：
- 投行、PE/VC最看重的能力
- 企业IPO怎么估值
- 并购重组怎么算对价

**三大核心问题**：
1. **投资决策**：公司应该投资什么样的长期资产
2. **融资决策**：公司如何筹集所需要的资金
3. **股利决策**：公司应该发放多少股利

**关键理论**：
- MM理论（资本结构与企业价值无关）
- CAPM模型（股权资本成本定价）
- 现金流折现法（DCF）

**实战应用**：
- 啃透1份IPO招股书
- 分析"融资用途"、"估值方法"、"财务数据"

#### 7. 金融工程学：量化时代的"技术钥匙"

**核心价值**：
- 用数学+编程解决金融问题
- 衍生品设计
- 风险对冲模型
- 量化策略回测

**关键工具**：
- Python量化交易
- 期权定价模型（Black-Scholes）
- 风险价值VaR模型

**实战应用**：
- 用Python写简单的均线策略
- 用Excel算期权定价

#### 8. 投资学：资产配置的"最优解算法"

**核心价值**：
- 在风险中赚取收益
- 建立"系统化投资框架"
- 而非靠"感觉"炒股

**关键理论**：
- 现代投资组合理论（MPT）
- 有效市场假说（EMH）
- 资本资产定价模型（CAPM）
- 夏普比率

**实战应用**：
- 做"模拟投资组合"
- 每月复盘收益与风险

#### 9. 金融计量学：金融研究的"数据分析器"

**核心价值**：
- 回归分析预测股价
- 时间序列模型分析波动
- 量化研究的必备工具

**关键工具**：
- Stata
- Python（pandas, statsmodels）
- EViews

#### 10. 财政学：政策影响的"解码器"

**核心价值**：
- 国债发行影响债券利率
- 税收政策改变企业盈利
- 政府投资拉动行业估值

**实战应用**：
- 跟踪年度财政预算报告
- 分析对国债市场、基建板块的影响

## 三、宏观经济指标深度解析

### 3.1 货币供应量（M0、M1、M2）

#### M0：流通中的"现金脉搏"

**定义**：流通中的现金，包括纸币和硬币

**特点**：
- 流动性最强
- 银行体系之外
- 不产生利息

**经济意义**：
- 春节前M0季节性骤增（取现备年货）
- M0持续异常增长可能暗示地下经济活跃
- 2023年中国M0占货币总量比例仅约4%

#### M1：企业活力的"晴雨表"

**定义**：M0 + 单位活期存款

**特点**：
- 代表全社会"即时购买力"
- 直接反映企业的经营活跃度
- 不包含个人储蓄存款

**经济意义**：
- 与经济周期关联性极强
- 与股市涨跌相关性高达70%
- 2020年疫情后M1增速从3%飙升至10%

#### M2：全社会的"货币池子"

**定义**：M1 + 单位定期存款 + 个人存款 + 其他存款

**特点**：
- 涵盖即时购买力和潜在购买力
- 央行调控的主要目标
- 2023年中国M2余额已达292万亿元

**关键警示**：
- M2与GDP比值过高可能引发资产泡沫
- 中国该比值约220%
- 日本1990年达290%后爆发房地产危机

### 3.2 剪刀差分析

#### M1-M2增速差（最核心指标）

**正剪刀差（M1 > M2）**：
- 企业愿意把定存转活期
- 准备扩大生产、增加投资
- 居民倾向消费而非储蓄
- **信号**：经济活跃、市场信心回升
- **历史案例**：2009年"四万亿"后M1大幅超过M2

**负剪刀差（M1 < M2）**：
- 企业宁愿存定期也不投入经营
- 居民倾向增加储蓄
- 资金陷入"沉睡"状态
- **信号**：经济收缩压力
- **历史案例**：2018年去杠杆时期

#### 社融-M2增速差

**含义**：
- 社融：从金融机构资产方出发（资金运用）
- M2：从金融机构负债方出发（资金供给）
- 理论上增速应该一致

**"剪刀差"含义**：
- 社融-M2 > 0：实体融资需求大于资金供给
- 社融-M2 < 0：资金供需不匹配，需求疲弱
- **出现时期**：2022年4月以来出现明显"剪刀差"

**实战应用**：
- 社融-M2增速差与10年国债收益率正相关
- 增速差走阔 → 资金价格走高 → 债券收益率上升
- 增速差收窄 → 资金供给快于需求 → 债券收益率下降

### 3.3 社会融资规模（社融）

#### 四大组成部分

1. **金融机构表内业务**
   - 人民币贷款（占比62.3%）
   - 外币贷款

2. **金融机构表外业务**
   - 委托贷款
   - 信托贷款
   - 未贴现银行承兑汇票

3. **直接融资**
   - 非金融企业境内股票
   - 企业债券（占比8.4%）

4. **其他项目**
   - 政府债券（占比18.3%）
   - 贷款核销
   - 保险公司赔偿

#### 贷款结构分析

**企业贷款**：
- 短期贷款：季节性明显，1月最高
- 票据融资：银行调节信贷量的工具
- 中长期贷款：与实体经济增长关联度高
  - 占比提高 → 经济复苏信号
  - 占比降低 → 企业投资意愿不足

**居民贷款**：
- 短期贷款：与汽车等消费相关
- 中长期贷款：以住房按揭贷款为主
  - 与房地产销售高度相关

## 四、投资学核心理论

### 4.1 现代投资组合理论（MPT）

**核心思想**：不要把所有鸡蛋放在一个篮子里

**关键概念**：
- **相关性**：不同资产一起涨跌的节奏
- **有效前沿**：风险一定时收益最高、收益一定时风险最低的组合
- **分散化**：通过科学搭配降低整体风险

**中国实例**：
```
正确组合：
├── 宁德时代（高增长、高波动的新能源）
├── 贵州茅台（稳健增长的消费）
├── 长江电力（低波动、高分红的公用事业）
└── 部分国债

效果：当宁德时代回调时，其他资产起到缓冲作用
```

### 4.2 资本资产定价模型（CAPM）

**核心公式**：
```
预期收益率 = 无风险利率 + β × (市场收益率 - 无风险利率)

ke = rf + β × (rm - rf)

ke：股权资本成本
rf：无风险利率（如国债收益率）
β：贝塔系数
rm：市场收益率
```

**贝塔系数的含义**：
- β > 1：波动大于市场（高风险高收益）
  - 实例：东方财富（牛市旗手）
- β < 1：波动小于市场（低风险低收益）
  - 实例：海天味业（稳健消费）

**关键洞察**：
- 系统性风险（β）可以被定价
- 非系统性风险可以通过分散化消除
- 市场只对系统性风险给予补偿

### 4.3 有效市场假说（EMH）

**三种形式**：
1. **弱式有效**：价格已反映所有历史信息
   - 技术分析无效
   - K线、金叉死叉无法预测未来

2. **半强式有效**：价格已反映所有公开信息
   - 基本面分析无效
   - 利好政策公布后股价瞬间到位

3. **强式有效**：价格已反映所有信息（包括内幕）
   - 内幕交易无法获利

**对投资的启示**：
- 承认市场的有效性
- 放弃"择时"和"抓牛股"的幻想
- 更理性的策略：低成本广泛持有市场本身（如定投沪深300ETF）

### 4.4 夏普比率

**公式**：
```
夏普比率 = (组合收益率 - 无风险利率) / 组合标准差
```

**含义**：
- 每承担一单位风险，获得的超额收益
- 越高越好
- 用于衡量"收益风险调整后的表现"

## 五、第一性原理深度应用

### 5.1 什么是第一性原理？

**定义**：穿透所有表象、套路和干扰，直击事物最核心、最本质、最基础的底层规律

**与类比思维的区别**：
- 类比思维：参考别人怎么做，在现有基础上改进
- 第一性原理：从0开始，基于基本事实推导

### 5.2 马斯克的4步操作法

#### 步骤1：拆解要素
- 将复杂问题分解成最基本的组成部分
- 实例：SpaceX火箭 → 分解为各个零部件和系统

#### 步骤2：追溯本源
- 深入探究每个要素背后的基础物理定律
- 实例：研究火箭推进原理、材料科学、空气动力学

#### 步骤3：重构系统
- 打破固有模式，重新设计
- 实例：可回收火箭设计（SpaceX成本降低90%）

#### 步骤4：验证迭代
- 不断对重构后的系统进行验证和改进
- 实例：每次发射都是验证机会

### 5.3 第一性原理在投资中的应用

#### 投资的本质问题

> **用今天的现金，换取未来更多的现金**

#### 三个核心问题

1. **这家公司未来能创造多少现金流？**
2. **这些现金流折现到今天值多少钱？**
3. **当前价格是否便宜？**

#### 思维转变

| 传统思维 | 第一性原理 |
|---------|-----------|
| "这只股票涨了很多" | "为什么涨？基本面变了还是情绪炒作？" |
| "RSI超买了要跌" | "RSI只是现象，真正的原因是什么？" |
| "跟着北向资金买" | "资金的决策依据是什么？" |
| "预测股价涨跌" | "评估企业价值，等待价格回归价值" |

### 5.4 第一性原理 vs 传统投资

| 维度 | 传统投资 | 第一性原理投资 |
|-----|---------|--------------|
| 分析重点 | 技术指标 | 企业本质 |
| 决策依据 | 历史规律、惯例 | 基本事实推导 |
| 风险控制 | 技术止损 | 估值安全边际 |
| 学习方向 | 更多技术指标 | 行业本质、商业模式 |
| 目标 | 预测涨跌 | 评估价值 |

## 六、公司金融核心理论

### 6.1 MM理论（莫迪利安尼-米勒）

**1958年提出**：
- 核心结论：在完美资本市场假设下，企业价值与资本结构无关
- 被称为"公司金融领域的牛顿定律"

**关键洞察**：
- 投资决策比融资决策更重要
- 企业价值取决于未来现金流
- 资本结构不影响企业价值（理想条件下）

### 6.2 现金流折现法（DCF）

**理论基础**：
- 费雪"现值"理论
- 资产价值 = 未来现金流量的折现值

**步骤**：
1. 预测未来5-10年的自由现金流
2. 确定折现率（通常用加权平均资本成本WACC）
3. 计算永续价值
4. 折现到今天
5. 得到企业价值

### 6.3 加权平均资本成本（WACC）

**公式**：
```
WACC = 股权比例 × 股权成本 + 债务比例 × 债务成本 × (1 - 税率)
```

**应用**：
- 作为DCF的折现率
- 评估企业资本结构是否最优

## 七、实战应用框架

### 7.1 宏观-中观-微观分析框架

#### 第一层：宏观分析（经济环境）

**关键问题**：
- 经济周期位置？（复苏/繁荣/衰退/萧条）
- 货币政策走向？（宽松/紧缩）
- 财政政策重点？（投资/消费）

**观察指标**：
- GDP增速
- PMI
- CPI/PPI
- M1-M2剪刀差
- 社融增速

#### 第二层：中观分析（行业选择）

**关键问题**：
- 哪些行业受益于当前宏观环境？
- 哪些行业符合经济发展趋势？
- 行业竞争格局如何？

**分析框架**：
- 行业生命周期
- 政策支持力度
- 竞争结构（波特五力）

#### 第三层：微观分析（公司选择）

**关键问题**：
- 公司竞争优势是什么？
- 未来能赚多少钱？
- 当前估值合理吗？

**分析维度**：
- 商业模式
- 护城河
- 财务质量
- 管理层

### 7.2 选股决策流程

```
第一步：本质分析（最重要）
├── 这家公司创造现金流的能力如何？
├── 行业前景和竞争优势？
└── 当前估值合理吗？
```

### 7.3 风险管理原则

**第一性原理**：投资的第一性原理是风控，而非暴利

**核心原则**：
1. **保住本金**：不懂止损、盲目跟风，哪怕赚十次，一次翻车就满盘皆输
2. **先控风险**：先控风险，再谈收益
3. **持续盈利**：不是博取暴利，而是保住本金、持续盈利

**具体措施**：
- 单只股票仓位不超过20%
- 设置硬性止损（如10%）
- 分散投资（不同行业、不同类型）
- 保留现金仓位
