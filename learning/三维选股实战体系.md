## 四、综合得分与推荐等级

### 4.1 综合得分计算

**总分** = 价值维度得分 + 技术维度得分 + 缠论维度得分

### 4.2 推荐等级

| 综合得分 | 推荐等级 | 说明 |
|---------|---------|-----|
| 80-100分 | **强烈推荐** | 三维都好，优先配置 |
| 60-79分 | **推荐** | 两维好，可以配置 |
| 40-59分 | **观望** | 一般，等待机会 |
| 0-39分 | **不推荐** | 不好，不参与 |

### 4.3 仓位建议

| 推荐等级 | 单只建议仓位 | 总仓位参考 |
|---------|------------|-----------|
| 强烈推荐 | 20-30% | 60-80%（强势市场） |
| 推荐 | 10-20% | 40-60%（震荡市场） |
| 观望 | 0-10% | 20-40%（弱势市场） |
| 不推荐 | 0% | 20%以下（不明朗） |

## 五、初选与精选流程

### 5.1 初选（必选条件）

| 检查项 | 是/否 | 说明 |
|-------|------|-----|
| 股价&lt;100元 | ☐ | 符合1万元本金 |
| 不是ST/*ST | ☐ | 回避风险股 |
| 没有退市风险 | ☐ | 回避风险股 |
| 不是问题股 | ☐ | 近期没有重大负面 |

**必须全部满足才能进入精选！**

---

### 5.2 精选（三维打分）

1. 价值维度打分（0-30分）
2. 技术维度打分（0-40分）
3. 缠论维度打分（0-30分）
4. 综合得分计算
5. 推荐等级确定

## 六、止损策略

### 6.1 根据买入类型设置止损

| 买入类型 | 止损位 | 说明 |
|---------|-------|-----|
| **第二类买点** | 买点级别中枢下沿下方5% | 最稳健，止损位明确 |
| **第一类买点** | 背驰点下方5-8% | 风险高，止损严格 |
| **第三类买点** | 中枢下沿下方3% | 突破确认，止损稍小 |

### 6.2 止损调整

- **盈利后**：可以提高止损位，保护利润
- **触及止损**：必须执行！不要侥幸！
- **重大利空**：立即止损或减仓！

## 八、重要原则

### 8.1 选股原则

1. **三维共振最好！**
   - 三维都好胜率最高
   - 两维好也可以
   - 一维好谨慎

2. **不要只看单一维度！**
   - 只看技术可能陷入炒作
   - 只看价值可能错过机会
   - 只看缠论可能忽略基本面

3. **宏观定方向！**
   - 先看宏观环境
   - 再选股

### 8.2 操作原则

1. **严格执行检查清单！**
2. **严格执行止损！**
3. **不要追高！**
4. **耐心等待机会！**

---

**使用方法**：选股时，先初选，再精选三维打分，最后确定推荐等级和仓位！
