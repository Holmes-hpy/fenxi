## 🔥 一、半导体产业链学习成果

### 1.1 行业核心认知

#### 战略地位
- 半导体已跃升为**支撑国家战略安全的核心带动型产业**
- "十五五"期间将从"摊大饼"模式转向**"锥体突破"**精准攻坚
- 国家大基金三期已集中落地，资金支持力度空前

#### 产业链全景图
```
上游：设备（光刻机、刻蚀机）+ 材料（光刻胶、电子特气）
     ↓
中游：芯片制造（晶圆代工）+ 封装测试
     ↓
下游：芯片设计 + 应用（消费电子、汽车、AI）
```

### 1.2 光刻胶产业链（重点关注⭐⭐⭐）

#### 2026年爆发逻辑
| 利好因素 | 具体表现 |
|---------|---------|
| 政策加码 | 工信部透露国产光刻胶最新进展 |
| 技术突破 | KrF/ArF高端光刻胶量产成功 |
| 需求爆发 | 下游晶圆厂扩产提速 |
| 国产替代 | 认证批量落地，缺货涨价周期启动 |

#### 投资机会
- **上游原材料**：树脂、溶剂、光引发剂
- **中游成品**：g/i线 → KrF → ArF → EUV（从易到难）
- **2026年重点**：ArF光刻胶量产放量

### 1.3 电子特气（新兴热点⭐⭐⭐）

#### 2026年5月新风口
- **玻璃基板**和**电子特气**站上C位
- 三重利好共振：供需紧张 + 国产替代 + 政策支持
- 国产替代已不是概念，进入兑现期

## 💡 二、绿电行业学习成果

### 2.1 行业核心数据

#### 光伏
- 中国光伏装机量**登顶全球**
- 绿电直连新模式启动，物理溯源明确

#### 风电
- 风电累计并网容量**占据世界一半份额**
- 海上风电成为新增长点

#### 储能（必修课⭐⭐⭐⭐⭐）
> "没储能 = 政策卡脖子 + 并网阻碍 + 盈利减半"

储能早已不是可选项，而是**行业的核心抓手**！

| 低碳基金 | 设立国家低碳转型基金 |

| 储能 | 加快新型储能发展 |
| 应用 | 推动绿电应用 |

### 2.3 投资机会排序

1. **⭐⭐⭐⭐⭐ 储能**：行业发展的瓶颈，也是最大机会
2. **⭐⭐⭐⭐ 海风**：风机大型化，海上风电加速

## 📈 四、投资策略总结

### 4.1 半导体投资框架
```
投资顺序：设备 → 材料 → 设计 → 制造
核心逻辑：国产替代 + 政策支持 + 业绩兑现
重点关注：光刻胶（2026爆发）+ 电子特气（新兴热点）
```

### 4.2 绿电投资框架
```
投资顺序：储能 → 运营 → 设备
核心逻辑：双碳目标 + 消纳解决 + 技术迭代
重点关注：储能（瓶颈即机会）+ 海风（大型化）
```

### 4.3 算力投资框架
```
投资顺序：芯片 → 服务器 → 应用
核心逻辑：AI发展 + 算力基建 + 自主可控
重点关注：AI芯片（ASIC机会）+ 光模块（数据传输）
```

## ⚠️ 五、风险提示

### 半导体
1. **美国出口管制升级**可能影响设备采购
2. **技术突破不及预期**可能导致追赶缓慢
3. **竞争加剧**可能导致价格战

### 绿电
1. **补贴退坡**影响运营商利润
2. **消纳问题**未完全解决
3. **原材料价格波动**影响中游利润

### 算力
1. **AI发展不及预期**影响算力需求
2. **算力过剩**可能导致价格下跌
3. **芯片封锁**影响高端芯片供应
