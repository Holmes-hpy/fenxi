## 五、核心知识点总结

### 知识点1：北向资金与市场走势相关性

**核心发现**：
- 当北向资金单日净流出超30亿元时，次日市场往往承压
- 本周北向净流出40.38亿元，上证指数下跌0.65%

**原理分析**：
```
北向资金 = 外资动向的"晴雨表"

流出原因：
- 人民币汇率波动
- 美股下跌，避险情绪
- 对A股短期看法偏空

影响机制：
- 外资重仓股（消费、金融）抛售
- 蓝筹股承压，影响指数
- 市场情绪受到压制

应对策略：
- 北向流出时，降低外资重仓股配置
- 增加内资主导的题材股配置
- 控制整体仓位
```

**实战应用**：
✅ 每天监测北向资金流向  
✅ 连续流出超50亿时，仓位降至50%以下  
✅ 关注沪股通vs深股通比例变化  

---

### 知识点2：行业轮动的"防御-进攻"规律

**核心发现**：
- 市场风险偏好下降时，资金从进攻板块（科技、消费）流向防御板块（煤炭、电力、银行）
- 本周煤炭(+3.93%)、电力(+1.47%)领涨，显示资金避险情绪

**原理分析**：
```
经济周期 → 行业轮动

复苏期（经济上行）：
- 进攻：消费、科技、周期
- 防御：减少配置

过热期（通胀上行）：
- 进攻：商品、原材料
- 防御：减少配置

滞胀期（经济下行+通胀）：
- 进攻：现金为王
- 防御：增加煤炭、电力、公用事业

衰退期（经济下行）：
- 进攻：债券、消费
- 防御：价值股、垄断行业
```

**实战应用**：
✅ 根据宏观经济周期调整行业配置  
✅ 滞胀期增加煤炭、电力配置  
✅ 复苏期增加消费、科技配置  

---

### 知识点3：AI算力产业链的"卖水人"逻辑

**核心发现**：
- 本周AI算力概念持续火爆，但不同环节表现分化
- PCB、光模块等"卖水人"比芯片制造更受追捧

**原理分析**：
```
AI算力产业链：

上游：芯片（GPU、ASIC）
  ❌ 风险：技术壁垒高，受制裁
  
中游：服务器、光模块、连接器
  ✅ 机会：技术相对成熟，国产替代快
  ✅ 机会：需求爆发，订单饱满
```
