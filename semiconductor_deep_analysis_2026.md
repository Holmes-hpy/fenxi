### 2.2 各环节深度分析

#### 2.2.1 光刻胶（Photoresist）

| 项目 | 内容 |
|------|------|
| **定义** | 光刻工艺中的关键感光材料，直接决定芯片制程精度，分g线(436nm)/i线(365nm)/KrF(248nm)/ArF(193nm)/EUV(13.5nm) |
| **技术壁垒** | 极高 - 核心配方、纯化技术、光源匹配 |
| **市场格局** | 日本JSR、东京应化(TOK)、富士胶片等垄断全球高端市场 |
| **国产现状** | 国产自给率不足10%，高端ArF/EUV光刻胶几乎为零 |
| **A股标的** | 彤程新材(603650)、晶瑞电材(300655)、雅克科技(002409) |
| **价格变化** | ArF进口涨20-25%，EUV光刻胶一季度涨30-35% |
| **投资价值** | ⭐⭐⭐⭐⭐ 高壁垒+高毛利+需求暴增 |

**产业链上下游关系：
```
上游（原材料） → 中游（合成/配方） → 下游（晶圆厂）
石油化工原料    → 光刻胶生产企业 → 芯片制造企业
（单体/溶剂    → 彤程新材/晶瑞电材 → 中芯国际/长江存储
```
